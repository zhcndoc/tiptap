构建一个符合您产品需求的富文本编辑器，Tiptap 用现代的、框架无关的 API 封装了经过验证的 ProseMirror 库，并为您提供各种扩展和功能。

## 什么是 Tiptap？

Tiptap 是一个无头富文本编辑器框架，允许您构建一个完全符合您和您的客户需求的自定义编辑器。
它基于 ProseMirror，一个经过实战考验的用于在网络上构建富文本编辑器的库。
在底层，Tiptap 大量依赖于 [事件](/editor/api/events)、[命令](/editor/api/commands) 和 [扩展](/editor/extensions/overview) 来提供灵活且强大的构建编辑器的 API。

## 为什么选择 Tiptap？

- **默认模块化：** 仅添加所需的扩展，从粗体和链接到复杂的表格和斜杠菜单。保持包的体积小，并控制您的架构。
- **无头优先：** 将 Tiptap 插入 React、Vue、Svelte、纯 JS 或任何其他框架中。或者与 React 的 [UI 组件和模板](/ui-components/getting-started/overview) 集成。
- **开源和 Pro 扩展：** Tiptap 的开源版本在 GitHub 上以 MIT 许可证发布。当您需要高级功能（如评论或 AI 命令）时，可以添加付费的 Tiptap 扩展。

想要开始使用 Tiptap？请参考我们的 [安装指南](/editor/getting-started/install)，用 Tiptap 设置您的第一个编辑器。

### 扩展您的编辑器

使用开源或 Pro 扩展扩展您的 Tiptap 编辑器。Tiptap 套件添加了更复杂的功能，并通过您的 [Tiptap Cloud 控制面板](https://cloud.tiptap.dev/v2/billing) 提供为期 30 天的免费试用。

- 从 [`StarterKit`](/editor/extensions/functionality/starterkit) 开始——一个包含核心标记和节点的全功能套件。
- 添加 [功能扩展](/editor/extensions/functionality)——字符计数器、占位符、历史记录。
- 添加 [内容扩展](/editor/extensions/overview)——图像、表格、图表、自定义节点。
- 通过查看我们的 [自定义扩展指南](/editor/extensions/custom-extensions) 创建您自己的扩展。
