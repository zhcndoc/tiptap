`ListKit` 是一个包含所有必要的 Tiptap 列表扩展的集合。如果你想快速在 Tiptap 中设置列表，这个扩展非常适合你。

## 安装

```bash
npm install @tiptap/extension-list@beta
```

## 包含的扩展

### 节点

- [`BulletList`](/editor/extensions/nodes/bullet-list)
- [`ListItem`](/editor/extensions/nodes/list-item)
- [`OrderedList`](/editor/extensions/nodes/ordered-list)
- [`TaskItem`](/editor/extensions/nodes/task-item)

## 使用 ListKit 扩展

将 `ListKit` 传递给编辑器以一次加载所有包含的扩展。

```js
import { Editor } from '@tiptap/core'
import { ListKit } from '@tiptap/extension-list'

const editor = new Editor({
  extensions: [ListKit],
})
```

你可以配置包含的扩展，甚至可以禁用其中的一些，如下所示。

```js
import { Editor } from '@tiptap/core'
import { ListKit } from '@tiptap/extension-list'

const editor = new Editor({
  extensions: [
    ListKit.configure({
      // 禁用扩展
      bulletList: false,

      // 配置扩展
      listItem: {
        HTMLAttributes: { class: 'list-item' },
      },
    }),
  ],
})
```
