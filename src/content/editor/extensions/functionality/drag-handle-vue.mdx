您是否曾希望在基于 Vue 的编辑器中拖动节点？好吧，我们也有这个想法——所以这就是为此而创建的扩展。

`DragHandleVue` 组件使您能够轻松地在编辑器中拖动节点。您可以定义自定义渲染函数、位置等。
它实质上将 [DragHandle](/editor/extensions/functionality/drag-handle) 扩展包装在 Vue 组件中，并会自动与编辑器注册/注销扩展。

## 属性

所有属性遵循与 [DragHandle](/editor/extensions/functionality/drag-handle) 扩展相同的结构。

### children

应该在拖拽句柄内显示的内容。

```vue
<drag-handle>
  <div>拖拽我！</div>
</drag-handle>
```

### tippyOptions

tippy.js 的选项。您可以传递在[tippy.js 文档](https://atomiks.github.io/tippyjs/v6/all-props/)中提供的任何选项。

默认值： `undefined`

```vue
<drag-handle :tippy-options="{ placement: 'left' }">
  <div>拖拽我！</div>
</drag-handle>
```

### onNodeChange

当节点被悬停时返回节点或 null。这可以用来高亮显示当前被悬停的节点。

默认值： `undefined`

```vue
<template>
  <drag-handle @nodeChange="handleNodeChange">
    <div>拖拽我！</div>
  </drag-handle>
</template>

<script>
import { ref } from 'vue'
import { DragHandle } from '@tiptap-pro/extension-drag-handle-vue-3'

export default {
  components: {
    DragHandle,
  },
  setup() {
    const selectedNode = ref(null)

    const handleNodeChange = ({ node, editor, pos }) => {
      selectedNode.value = node
      // 对节点执行某些操作
    }

    return {
      selectedNode,
      handleNodeChange,
    }
  },
}
</script>
```

### pluginKey

应在编辑器中用于存储插件的键。如果您在同一编辑器中有多个拖拽句柄，这非常有用。

默认值： `undefined`

```vue
<drag-handle pluginKey="myCustomDragHandle">
  <div>拖拽我！</div>
</drag-handle>
```
