虽然 [节点视图](/editor/extensions/custom-extensions/node-views) 用于渲染节点，但标记视图用于渲染标记。它们相对简单，因为它们不必处理节点的复杂性，并且提供了更简单的 API。

标记视图极大地改善了编辑器内的体验，但也可以在 Tiptap 的只读实例中使用。它们设计上与 HTML 输出无关，因此你可以完全控制编辑器内的体验 _和_ 输出。

### 渲染 JavaScript/Vue/React

但如果你想渲染实际的 JavaScript/Vue/React 代码怎么办？考虑使用 Tiptap 来渲染你的输出。只需将编辑器设置为 `editable: false`，没有人会注意到你在使用编辑器来渲染内容。 :-)
