该标记呈现一个 `<span>` HTML 标签，并允许您添加一系列与样式相关的属性，例如字体系列、字体大小或颜色。该扩展默认不添加任何样式属性，但其他扩展将其作为基础，例如 [`FontFamily`](/editor/extensions/functionality/fontfamily) 或 [`Color`](/editor/extensions/functionality/color)。

## 安装

```bash
npm install @tiptap/extension-text-style@beta
```

## 命令

### removeEmptyTextStyle()

移除没有内联样式的 `<span>` 标签。

```js
editor.command.removeEmptyTextStyle()
```
