代码扩展使您可以在编辑器中使用 `<code>` HTML 标签。如果您粘贴带有 `<code>` 标签的文本，它将相应地呈现。

输入一些带有 <code>\`反引号\`</code> 的内容，它会在您输入时神奇地转换为 `内联代码`。

<CodeDemo path="/Marks/Code" />

## 安装

```bash
npm install @tiptap/extension-code@beta
```

## 设置

### HTMLAttributes

自定义 HTML 属性，应添加到渲染的 HTML 标签中。

```js
Code.configure({
  HTMLAttributes: {
    class: 'my-custom-class',
  },
})
```

## 命令

### setCode()

将文本标记为内联代码。

```js
editor.commands.setCode()
```

### toggleCode()

切换内联代码标记。

```js
editor.commands.toggleCode()
```

### unsetCode()

移除内联代码标记。

```js
editor.commands.unsetCode()
```

| 命令          | Windows/Linux                     | macOS                         |
