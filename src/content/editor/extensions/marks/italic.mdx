使用此扩展以 _斜体_ 形式呈现文本。如果您在编辑器的初始内容中传递 `<em>`、`<i>` 标签或带有 `style` 属性设置为 `font-style: italic` 的文本，它们都将被相应地渲染。

输入 `*一个星号*` 或 `_一个下划线_`，当您输入时它将神奇地转换为 _斜体_ 文本。

<CodeDemo path="/Marks/Italic" />

<Callout title="限制" variant="default">
    该扩展将在读取 `Editor` 实例的内容时生成相应的 `<em>` HTML 标签。所有标记为斜体的文本，无论使用何种方法，都将被规范化为 `<em>` HTML 标签。
</Callout>

## 安装

```bash
npm install @tiptap/extension-italic@beta
```

## 设置

### HTMLAttributes

应该添加到渲染的 HTML 标签中的自定义 HTML 属性。

```js
Italic.configure({
  HTMLAttributes: {
    class: 'my-custom-class',
  },
})
```

## 命令

### setItalic()

将文本标记为斜体。

```js
editor.commands.setItalic()
```

### toggleItalic()

切换斜体标记。

```js
editor.commands.toggleItalic()
```

### unsetItalic()

移除斜体标记。

```js
editor.commands.unsetItalic()
```
