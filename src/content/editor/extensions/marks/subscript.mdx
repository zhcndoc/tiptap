使用此扩展在 <sub>下标</sub> 中渲染文本。如果您在编辑器的初始内容中传递 `<sub>` 或带有 `vertical-align: sub` 的文本作为内联样式，两者都会被相应渲染。

<Callout title="限制" variant="default">
    当读取 `Editor` 实例的内容时，扩展会生成相应的 `<sub>` HTML 标签。所有下标文本，无论使用何种方法，都会被规范化为 `<sub>` HTML 标签。
</Callout>

## 安装

```bash
npm install @tiptap/extension-subscript@beta
```

## 设置

### HTMLAttributes

应添加到渲染的 HTML 标签的自定义 HTML 属性。

```js
Subscript.configure({
  HTMLAttributes: {
    class: 'my-custom-class',
  },
})
```

## 命令

### setSubscript()

将文本标记为下标。

```js
editor.commands.setSubscript()
```

### toggleSubscript()

切换下标标记。

```js
editor.commands.toggleSubscript()
```

### unsetSubscript()

移除下标标记。

```js
editor.commands.unsetSubscript()
```
