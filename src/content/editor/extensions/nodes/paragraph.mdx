## 安装

```bash
npm install @tiptap/extension-paragraph@next
```

## 设置

### HTMLAttributes

应添加到渲染的 HTML 标签的自定义 HTML 属性。

```js
Paragraph.configure({
  HTMLAttributes: {
    class: 'my-custom-class',
  },
})
```

## 命令

### setParagraph()

将所有选定的节点转换为段落。

```js
editor.commands.setParagraph()
```
