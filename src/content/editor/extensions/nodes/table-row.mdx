没有行的表格是什么？添加这个扩展让您的表格可用。

<CodeDemo path="/Nodes/Table" />

## 安装

```bash
npm install @tiptap/extension-table@beta
```

此扩展需要安装 [`Table`](/editor/extensions/nodes/table) 扩展。

该扩展默认与 `TableKit` 扩展一起安装，因此您无需单独安装。

```ts
import { Editor } from '@tiptap/core'
import { TableKit } from '@tiptap/extension-table'

new Editor({
  extensions: [TableKit],
})
```
