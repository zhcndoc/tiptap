`Text` 扩展是必需的，至少如果你想处理任何类型的文本，这一点非常可能。这个扩展有些不同，它甚至不渲染 HTML。它就是纯文本，仅此而已。

<Callout title="重大变化" variant="info">
  Tiptap v1 尝试将该节点隐藏，但它一直存在。从现在起，你必须显式导入它（或使用 [StarterKit](/editor/extensions/functionality/starterkit)）。
</Callout>

<CodeDemo path="/Nodes/Text" />

## 安装

```bash
npm install @tiptap/extension-text@next
```
