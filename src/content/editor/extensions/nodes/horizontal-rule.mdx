使用这个扩展来渲染 `<hr>` HTML 标签。如果你在编辑器的初始内容中传入 `<hr>`，它会相应地被渲染出来。

在新行的开头输入三个破折号（<code>---</code>）或者三个下划线加空格（<code>\_\_\_ </code>），它们会神奇地变成一条横线。

<CodeDemo path="/Nodes/HorizontalRule" />

## 安装

```bash
npm install @tiptap/extension-horizontal-rule@beta
```

## 设置

### HTMLAttributes

自定义将添加到渲染HTML标签中的属性。

```js
HorizontalRule.configure({
  HTMLAttributes: {
    class: 'my-custom-class',
  },
})
```

## 命令

### setHorizontalRule()

创建一条水平线。

```js
editor.commands.setHorizontalRule()
```
