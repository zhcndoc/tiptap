无论您使用 Tiptap 构建什么，`Document` 扩展都是必需的。它是一个所谓的“顶级节点”，是所有其他节点的家。可以将其视为您文档的 `<body>` 标签。

但是这个节点非常小。它定义了节点的名称（`doc`），配置为顶级节点（`topNode: true`），并且可以包含多个其他节点（`block+`）。就这些。您可以自己看看：

<CodeDemo path="/Nodes/Document" />

<Callout title="重大变更" variant="warning">
  Tiptap v1 尝试向您隐藏该节点，但它始终存在。从现在开始，您必须显式导入它（或者使用 [StarterKit](/editor/extensions/functionality/starterkit)）。
</Callout>

## 安装

```bash
npm install @tiptap/extension-document
```
