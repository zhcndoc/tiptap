这是一个用于处理 [Tiptap](https://tiptap.dev/) 内容的 PHP 包。你可以将 Tiptap 兼容的 JSON 转换为 HTML，反之亦然，进行内容 sanitization，或只是修改内容。

## 安装

你可以通过 Composer 安装该包：

```bash
composer require ueberdosis/tiptap-php
```

## 使用 Tiptap PHP 工具

这个 PHP 包模仿了 JavaScript 版本的大部分功能。如果你熟悉 Tiptap，PHP 的语法也会让你感觉很熟悉。下面是一个简单的示例：

```php
(new Tiptap\Editor)
    ->setContent('<p>示例文本</p>')
    ->getDocument();

// 返回：
// ['type' => 'doc', 'content' => …]
```
