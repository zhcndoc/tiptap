`joinForward` 命令用于将当前选择的两个节点向前合并。如果选择为空且位于文本块的末尾，`joinForward` 将尝试减少该块与它之后的块之间的距离。[另见](https://prosemirror.net/docs/ref/#commands.joinForward)

## 使用 joinForward 命令

```js
editor.commands.joinForward()
```
