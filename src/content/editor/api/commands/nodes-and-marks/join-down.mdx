`joinDown` 命令将选定的块，或者如果有文本选择，则将选择的最近祖先块（可以连接）与其下方的同级块连接在一起。[另请参阅](https://prosemirror.net/docs/ref/#commands.joinDown)

## 使用 joinDown 命令

```js
editor.commands.joinDown()
```
