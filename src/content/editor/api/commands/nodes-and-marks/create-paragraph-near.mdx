如果当前选中了一个块节点，`createParagraphNear` 命令将在当前选中的块节点之后创建一个空段落。如果选中的块节点是其父节点的第一个子节点，则新段落将插入到当前选择之前。

## 使用 createParagraphNear 命令

```js
editor.commands.createParagraphNear()
```
