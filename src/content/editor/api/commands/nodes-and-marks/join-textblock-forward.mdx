joinForward 的更受限版本，仅在光标位于文本块末尾时，尝试将当前文本块与下一个文本块合并。[另请参阅](https://prosemirror.net/docs/ref/#commands.joinTextblockForward)

## 使用 joinTextblockForward 命令

```js
editor.commands.joinTextblockForward()
```
