`splitBlock` 会在当前 [NodeSelection](https://prosemirror.net/docs/ref/#state.NodeSelection) 的位置将当前节点分割为两个节点。如果当前选择不可分割，则此命令将被忽略。

## 参数

`options: Record<string, any>`

- `keepMarks: boolean` - 定义是否保留标记或移除标记。默认值为 `true`。

## 使用 splitBlock 命令

```js
// 分割当前节点并保留标记
editor.commands.splitBlock()

// 分割当前节点并不保留标记
editor.commands.splitBlock({ keepMarks: false })
```
