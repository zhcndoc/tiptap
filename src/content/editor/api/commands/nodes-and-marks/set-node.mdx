`setNode` 命令将用给定节点替换给定范围。范围取决于当前的选择。**重要**：目前 `setNode` 仅支持文本块节点。

## 参数

`typeOrName: string | NodeType`

将替换范围的节点类型。可以是字符串或 NodeType。

`attributes?: Record<string, any>`

应该应用于节点的属性。**这不是必需的。**

## 使用 setNode 命令

```js
editor.commands.setNode('paragraph', { id: 'paragraph-01' })
```
