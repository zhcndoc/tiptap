`clearNodes` 命令将节点标准化为默认节点，默认情况下为段落。它甚至可以标准化各种列表。在一些高级用例中，在应用新节点类型之前使用它会很有帮助。

如果你想知道如何定义默认节点：这取决于你的 [`Document`](/editor/extensions/nodes/document) 的 `content` 属性，默认情况下是 `block+`（至少一个块节点），而 [`Paragraph`](/editor/extensions/nodes/paragraph) 节点的优先级最高，因此它会首先加载，因此成为默认节点。

## 使用 clearNodes 命令

```js
editor.commands.clearNodes()
```
