`joinBackward` 命令从当前选择向后连接两个节点。如果选择为空并且位于文本块的开头，`joinBackward` 将尝试减少该块与其前面的块之间的距离。[参见](https://prosemirror.net/docs/ref/#commands.joinBackward)

## 使用 joinBackward 命令

```js
editor.commands.joinBackward()
```
