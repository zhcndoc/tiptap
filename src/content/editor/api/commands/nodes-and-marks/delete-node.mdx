`deleteNode` 命令删除当前选区内的一个节点。它需要一个 `typeOrName` 参数，这可以是一个字符串或一个 `NodeType`，用于查找需要删除的节点。删除节点后，视图将自动滚动到光标的位置。

## 参数

`typeOrName: string | NodeType`

## 使用 deleteNode 命令

```js
// 删除一个段落节点
editor.commands.deleteNode('paragraph')

// 或者

// 删除一个自定义节点
editor.commands.deleteNode(MyCustomNode)
```
