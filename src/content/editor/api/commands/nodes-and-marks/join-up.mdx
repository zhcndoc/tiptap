`joinUp` 命令将所选块，或者如果有文本选中，则将可以合并的选区最近的祖先块，与其上方的兄弟块合并。[另见](https://prosemirror.net/docs/ref/#commands.joinUp)

## 使用 joinUp 命令

```js
editor.commands.joinUp()
```
