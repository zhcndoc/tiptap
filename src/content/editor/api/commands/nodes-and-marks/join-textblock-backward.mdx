一种更有限的 joinBackward 形式，仅在光标位于文本块开头时尝试将当前文本块与前一个文本块连接。[另请参阅](https://prosemirror.net/docs/ref/#commands.joinTextblockBackward)

## 使用 joinTextblockBackward 命令

```js
editor.commands.joinTextblockBackward()
```
