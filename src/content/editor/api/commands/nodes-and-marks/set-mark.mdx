`setMark` 命令将在当前位置添加一个新的标记。

## 参数

`typeOrName: string | MarkType`

要添加的标记类型。可以是字符串或 MarkType。

`attributes: Record<string, any>`

应应用于该标记的属性。**这是可选的。**

## 使用 setMark 命令

```js
editor.commands.setMark('bold', { class: 'bold-tag' })
```
