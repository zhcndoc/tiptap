列表是在 Tiptap 编辑器中构建内容的重要部分。Tiptap 提供命令来轻松操作列表结构。以下是一些帮助您创建、更新和管理列表的基本命令概述。

## 用例

- **创建和切换列表：** 使用 [`toggleList`](/editor/api/commands/lists/toggle-list) 创建或在列表类型之间切换。
- **嵌套和去嵌套列表项：** 使用命令如 [`liftListItem`](/editor/api/commands/lists/lift-list-item) 和 [`sinkListItem`](/editor/api/commands/lists/sink-list-item) 举起或放下列表项。
- **分割和包装列表项：** 利用 [`splitListItem`](/editor/api/commands/lists/split-list-item) 和 [`wrapInList`](/editor/api/commands/lists/wrap-in-list) 高效地分割或包装列表项。
- **改善列表键盘行为：** 使用 [`List Keymap`](/editor/extensions/functionality/listkeymap) 扩展，通过额外的键映射处理程序来精细化列表行为。

## 列表键映射扩展

您可能还想包含 [`List Keymap`](/editor/extensions/functionality/listkeymap) 扩展，该扩展增加额外的键映射处理程序，以更改列表的默认退格和删除行为。它修改了默认行为，使得在列表项开头按下退格键可以将内容提升到上面的列表项中。

## 这是… 列表命令的列表

| 命令            | 描述                                                                                       |
| --------------- | ------------------------------------------------------------------------------------------ |
| `liftListItem`  | 尝试将当前选中的列表项提升到一个包装的父列表中。                                          |
| `sinkListItem`  | 将当前选中的列表项沉入到一个包装的子列表中。                                            |
| `splitListItem` | 将一个列表项分割成两个独立的列表项。                                                    |
| `toggleList`    | 在不同类型的列表之间切换。                                                                |
| `wrapInList`    | 将当前选中的节点包装在一个列表中。                                                       |
