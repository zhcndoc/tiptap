`splitListItem` 将一个列表项拆分为两个独立的列表项。如果这是一个嵌套列表，则应拆分外层的列表项。

## 参数

`typeOrName: string | NodeType`

应该被拆分为两个独立列表项的节点类型。

## 使用 splitListItem 命令

```js
editor.commands.splitListItem('bulletList')
```
