使用这些命令可以动态插入、替换或移除编辑器中的内容。初始化新文档、更新现有文档或管理用户选择，这些命令为您提供了处理内容操作的工具。

## 使用案例

- **初始化新文档：** 使用 [`setContent`](/editor/api/commands/content/set-content) 命令从头开始初始化一个干净的文档或预定义模板。
- **更新现有内容：** 使用 [`insertContent`](/editor/api/commands/content/insert-content) 或 [`insertContentAt`](/editor/api/commands/content/insert-content-at) 命令根据用户交互添加新内容或更新特定部分。
- **清除内容：** 使用 [`clearContent`](/editor/api/commands/content/clear-content) 命令移除所有内容，同时保持有效的文档结构。
- **管理用户选择：** 根据用户选择，使用 [`insertContentAt`](/editor/api/commands/content/insert-content-at) 插入或替换特定位置或范围的内容。

## 内容命令列表

| 命令               | 描述                                                               |
| ----------------- | ----------------------------------------------------------------- |
| `clearContent`    | 删除当前文档，同时遵循编辑器的架构。                             |
| `insertContent`   | 使用纯文本、HTML 或 JSON 向文档添加内容。                        |
| `insertContentAt` | 在文档中的特定位置或范围插入内容。                              |
| `setContent`      | 使用 JSON 或 HTML 替换整个文档为新文档。                       |
