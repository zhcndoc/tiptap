该命令剪切内容并将其放置到指定位置。

另请参见: [focus](/editor/api/commands/selection/focus)

## 使用 cut 命令

```js
const from = editor.state.selection.from
const to = editor.state.selection.to

const endPos = editor.state.doc.nodeSize - 2

// 从范围中剪切内容并将其放置在文档的末尾
editor.commands.cut({ from, to }, endPos)
```
