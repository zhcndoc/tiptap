在当前事务中存储一个元数据属性。

## 参数

`key: string`

元数据的名称。您可以随时使用 [getMeta](https://prosemirror.net/docs/ref/#state.Transaction.getMeta) 获取其值。

`value: any`

在您的元数据中存储任何值。

## 使用 setMeta 命令

```js
// 防止触发更新事件
editor.commands.setMeta('preventUpdate', true)

// 在当前事务中存储任何值。
// 您可以随时使用 tr.getMeta('foo') 获取此值。
editor.commands.setMeta('foo', 'bar')
```
