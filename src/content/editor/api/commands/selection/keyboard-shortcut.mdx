`keyboardShortcut` 命令将尝试触发一个具有给定名称的 ShortcutEvent。

## 参数

`name: String`

要触发的快捷方式名称。

## 使用 keyboardShortcut 命令

```js
editor.commands.keyboardShortcut('undo')
```
