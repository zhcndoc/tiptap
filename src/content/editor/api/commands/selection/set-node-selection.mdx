`setNodeSelection` 在给定位置创建一个新的 NodeSelection。节点选择是指向单个节点的选择。 [查看更多](https://prosemirror.net/docs/ref/#state.NodeSelection)

## 参数

`position: number`

NodeSelection 将在此位置创建。

## 使用 setNodeSelection 命令

```js
editor.commands.setNodeSelection(10)
```
