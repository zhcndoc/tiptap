Tiptap Editor 的基础 AI 生成扩展包含一组预配置命令，您可以将其集成到富文本编辑器中。这些命令可让您调整文本语气、补全文本、生成图片等，从而增强编辑器的功能。

要了解这些命令的使用方式，请查看 [概览](/ai/basic/overview) 页面上的示例。

| 命令                                                                                                                         | 描述                                                                                                                          |
| ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| [`aiAdjustTone(tone: Tone, options: TextOptions)`](#aiadjusttone)                                                               | 将所选文本的语气调整为指定的 [TONE](#aitone)。                                                     |
| [`aiBloggify(options: TextOptions)`](#most-text-commands)                                                                       | 将文本重写为博客格式                                                                                                  |
| [`aiComplete(options: TextOptions)`](#most-text-commands)                                                                       | 补全所选文本                                                                                                          |
| [`aiDeEmojify(options: TextOptions)`](#most-text-commands)                                                                      | 移除所选文本中的表情符号                                                                                                |
| [`aiEmojify(options: TextOptions)`](#most-text-commands)                                                                        | 为您的文本添加表情符号 ✨                                                                                                          |
| [`aiExtend(options: TextOptions)`](#most-text-commands)                                                                         | 扩展您的文本                                                                                                                    |
| [`aiFixSpellingAndGrammar(options: TextOptions)`](#most-text-commands)                                                          | 修正拼写和语法                                                                                                             |
| [`aiKeypoints(options: TextOptions)`](#most-text-commands)                                                                      | 将您的文本总结为要点列表                                                                                         |
| [`aiRephrase(options: TextOptions)`](#most-text-commands)                                                                       | 改写所选文本                                                                                                          |
| [`aiRestructure(options: TextOptions)`](#most-text-commands)                                                                    | 将所选文本重构为使用富文本格式                                                                           |
| [`aiShorten(options: TextOptions)`](#most-text-commands)                                                                        | 缩短所选文本                                                                                                           |
| [`aiSimplify(options: TextOptions)`](#most-text-commands)                                                                       | 用更简洁的词语改写您的文本                                                                                              |
| [`aiSummarize(options: TextOptions)`](#most-text-commands)                                                                      | 总结您的文本                                                                                                                 |
| [`aiTextPrompt(options: TextOptions)`](#most-text-commands)                                                                     | 运行您的自定义提示词                                                                                                              |
| [`aiTldr(options: TextOptions)`](#most-text-commands)                                                                           | 创建一个“Too Long; Didn't Read”版本的文本                                                                                       |

### 大多数文本命令

大多数文本命令接受相同的选项，使用方法类似。以下示例展示了如何使用 `aiBloggify` 命令：

```js
editor.chain().focus().aiBloggify(options: TextOptions)
```

### aiAdjustTone

```js
// 语气: 'default' | 'academic' | 'business' | 'casual' | 'childfriendly' | 'confident' | 'conversational' | 'creative' | 'emotional' | 'excited' | 'formal' | 'friendly' | 'funny' | 'humorous' | 'informative' | 'inspirational' | 'memeify' | 'narrative' | 'objective' | 'persuasive' | 'poetic' | string
editor.chain().focus().aiAdjustTone(tone: Tone, options: TextOptions).run()
```

### aiTranslate

将选中的文本内容翻译为指定的输出语言。

接受两个字母 ISO 639-1 语言代码。

```js
// 语言: 'en' | 'de' | 'nl' | ...
editor.chain().focus().aiTranslate(language: Language, options: TextOptions).run()
```

## 文本命令选项

对于支持 `TextOptions` 的每个命令，您可以指定以下选项：

| Setting          | Type                                                                   | Default                                       | Definition                                                                                                                                                                                                                 |
| ---------------- | ---------------------------------------------------------------------- | --------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `modelName`      | see [支持的文本模型](#支持的文本模型)                    | `gpt-5-mini` (`gpt-4.1` if `streaming: true`) | 在 OpenAI 中使用的模型                                                                                                                                                                                                   |
| `format`         | `'rich-text' \| 'plain-text'`                                          | `plain-text`                                  | 决定模型应如何响应的 [格式](/ai/basic/text-generation/format)                                                                                                        |
| `stream`         | `boolean`                                                              | `false`                                       | 命令是否应将字符流式输出到编辑器？这类似于 ChatGPT 中的 **打字机** 效果。**这需要最新的扩展版本！**                                                                      |
| `insertAt`       | `undefined \| number \| {from: number; to: number}`                    | `undefined`                                   | 将响应插入到编辑器中的位置；如果为 `undefined`，响应将插入到当前选区。如果是数字，响应将插入到该位置。如果是对象，则会替换该范围。 |
| `language`       | `string` (e.g. `en`, `de`)                                             | `null`                                        | 虽然我们会尽力提示 OpenAI 以输入语言生成响应，但有时最好由您自己定义。                                                                                         |
| `tone`           | `string`                                                               | `null`                                        | 响应应转换为的 [语气](#ai-adjust-tone)                                                                                                                                                   |
| `textLength`     | `number`                                                               | `undefined`                                   | AI 应响应的 `textLengthUnit` 数量（例如，“3 段”中的 `3`）                                                                                                                                |
| `textLengthUnit` | `'paragraphs' \| 'words' \| 'characters'`                              | `undefined`                                   | AI 应响应的 `textLength` 单位（例如，“3 段”中的 `paragraphs`）                                                                                                                             |
| `collapseToEnd`  | `boolean`                                                              | `true`                                        | 操作完成后光标是否应设置到末尾，或者是否应选中插入内容。                                                                                                                      |
| `context`        | `Array<{ type: 'text', text: string} \| { type: 'url', url: string }>` | `[]`                                          | 提供给模型用于生成响应的 [上下文](/ai/basic/text-generation/provide-context)。                                                                                             |
| `text`           | `string`                                                               | `undefined`                                   | 供模型响应的可选消息                                                                                                                                                                           |
| `insert`         | `boolean`                                                              | `true`                                        | 允许禁用直接将内容插入编辑器                                                                                                                                                                |
| `append`         | `boolean`                                                              | `true`                                        | 如果为 `true`，响应将追加到当前选区末尾。如果为 `false`，响应将替换当前选区。                                                                                 |
| `regenerate`     | `boolean`                                                              | `false`                                       | 命令是否重新生成之前生成的响应。如果设置为 true，`initialContent` 属性将不会更新。                                                                             |

_遗憾的是，**语气**与**语言**的组合有时会导致生成的响应不符合预期语言。_

## 选择合适的模型

在配置 Tiptap AI 扩展时，请根据您的应用需求考虑：

- **对于成本效益型操作：** 如果主要关注预算，且任务不需要最先进的能力，则选择 GPT-5 mini。
- **对于高级需求：** 如果您的应用需要最高水平的语言理解或图像生成质量，且预算限制较少，则选择 GPT-5。

Tiptap AI 扩展的灵活配置使您能够根据具体需求和预算调整 AI 集成方案。
