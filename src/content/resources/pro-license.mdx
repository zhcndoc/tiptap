Tiptap GmbH 授予您持续的、非排他性的许可，以使用 Tiptap Pro 扩展，前提是您仍然是 Tiptap Pro 的活跃订阅用户。

如果您订阅了免费计划，则该许可证允许**一位个人**（被许可人）访问和使用 Tiptap Pro 扩展。

如果您购买了入门计划，则该许可证允许**被许可人的最多2名员工和承包商**访问和使用 Tiptap Pro 扩展。

如果您购买了商业计划，则该许可证允许**被许可人的最多8名员工和承包商**访问和使用 Tiptap Pro 扩展。

您**可以**：

+ 使用 Tiptap Pro 扩展创建无限数量的最终产品。
+ 使用 Tiptap Pro 扩展为无限客户创建最终产品。
+ 使用 Tiptap Pro 扩展创建的最终产品并将其销售给终端用户。
+ 使用 Tiptap Pro 扩展创建开源且免费提供给终端用户的最终产品。

您**不能**：

- 使用 Tiptap Pro 扩展创建允许终端用户使用 Tiptap Pro 扩展或其衍生品自行构建最终产品的最终产品。
- 修改 Tiptap Pro 扩展以创建衍生的 Tiptap 扩展。
- 分发 Tiptap Pro 扩展或其衍生品，且不作为最终产品的一部分。
- 在订阅免费计划的情况下，将您的账户或对 Tiptap Pro 扩展的访问权限分享给其他个人。
- 使用 Tiptap Pro 扩展创建归任何个人或实体——除被许可人或被许可人客户以外的——所有权的最终产品。
- 使用 Tiptap Pro 扩展制造任何在 Tiptap GmbH 独自酌情判断下，被视为与 Tiptap GmbH 业务竞争或串通的产品。

## 示例用法
被许可证**允许**的使用示例：

+ 自己创建个人网站或网页应用。
+ 为客户创建网站或网页应用，最终归该客户所有。
+ 创建商业型 SaaS 应用（例如开票软件），终端用户需付费使用。
+ 创建以付费方式销售给终端用户的商业自托管网页应用。
+ 创建其主要目的并非分享 Tiptap Pro 扩展的网页应用（如使用 Tiptap Pro 扩展作为编辑器的会议组织应用），该应用免费、开源，源代码公开。

被许可证**不允许**的使用示例：

- 创建收藏的 Tiptap Pro 扩展（或其衍生品）仓库并公开发布。
- 重新使用其他编程语言或程序库重建 Tiptap Pro 扩展，且提供出售或免费使用。
- 创建“编辑器构建器”项目，让终端用户可以使用 Tiptap Pro 扩展或其衍生品自行构建编辑器。
- 创建任何非由您、您的公司或您的公司的客户唯一拥有的最终产品。例如，您的员工/承包商不得使用贵公司的 Tiptap Pro 许可证来构建他们自己的网站或副项目。

## 免费（单个开发者）定义
被许可人是订阅免费计划的个人。

Tiptap Pro 扩展是指在订阅免费计划后提供给被许可人的源代码。

最终产品是指结合 Tiptap Pro 扩展或其衍生品生成的任何工件。

终端用户是最终产品的用户。

客户是直接由被许可人提供定制专业服务的个人或实体，为其专门制作。软件即服务（SaaS）产品的客户在本文件中不视为客户。

## 入门或商业许可证的定义
被许可人是购买了入门或商业许可证的企业实体。

Tiptap Pro 扩展是指在购买入门或商业许可证后提供给被许可人的源代码。

最终产品是指结合 Tiptap Pro 扩展或其衍生品生成的任何工件。

终端用户是最终产品的使用者。

员工是被许可人的全职或兼职员工。

承包商是受雇为被许可人提供服务的个人或企业实体。

客户是直接由被许可人提供定制专业服务的个人或实体，为其专门制作。软件即服务（SaaS）产品的客户在本文件中不视为客户。
