一个预构建的[代理技能](https://agentskills.io/)，教会 AI 编程代理如何集成和使用 Tiptap。

该技能为你的代理提供：

- 搜索 Tiptap 源代码和文档的策略。
- 使用 Tiptap 3 的最佳实践
- 特定框架的指导（例如 React）

## 安装

使用 [skills CLI](https://skills.sh/) 安装该技能：

```bash
npx skills add ueberdosis/tiptap
```

或者[直接从 GitHub 下载 SKILL.md](https://github.com/ueberdosis/tiptap/blob/main/skills/tiptap/SKILL.md)。
