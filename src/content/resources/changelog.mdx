Tiptap 由超过 50 个独立的包组成。以下是您需要了解更改的所有信息：

1. 查看 [GitHub 发行页面](https://github.com/ueberdosis/tiptap/releases) 以获取最佳概述。
2. 更多详细信息，请访问 [Tiptap 在 GitHub 上的 CHANGELOG](https://github.com/ueberdosis/tiptap/blob/main/CHANGELOG.md)。
3. 此外，所有包也有 [单独的 `CHANGELOG.md`](https://github.com/ueberdosis/tiptap/blob/main/packages)。
4. 关注 [GitHub 上的合并 Pull Requests](https://github.com/ueberdosis/tiptap/pulls?q=is%3Apr+is%3Aclosed) 也很有帮助。

[我们会撰写博客文章](https://tiptap.dev/blog) 以介绍重要更改，并在此处保持更新。
