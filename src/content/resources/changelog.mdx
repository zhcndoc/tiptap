Tiptap 包含 50 多个独立的软件包。请使用侧边栏中生成的软件包页面来获取软件包级别的更新信息，并首先查看根目录下的 changelog 页面以获得最全面的概览。

以下是你需要关注的更新信息：

1. 查看 [GitHub 发布页面](https://github.com/ueberdosis/tiptap/releases) 以获取最佳概览。
2. 了解更多细节，请访问 [Tiptap 的 CHANGELOG](https://github.com/ueberdosis/tiptap/blob/main/CHANGELOG.md)。
3. 此外，每个软件包都有其独立的生成式 changelog 页面，这些页面基于上游的 `CHANGELOG.md` 文件。
4. 关注 [GitHub 上的已合并 Pull Requests](https://github.com/ueberdosis/tiptap/pulls?q=is%3Apr+is%3Aclosed) 也很有帮助。

[我们会撰写博客文章](https://tiptap.dev/blog) 以介绍重要更改，并在此处保持更新。
