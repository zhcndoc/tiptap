<a href="https://tiptap.dev/contact-sales?form=ai-toolkit">联系我们团队</a>购买付费的 AI 工具包附加组件。获得访问权限后，按照[私有注册表指南](/guides/pro-extensions)配置你的包管理器。

然后，安装该包：

```bash
npm install @tiptap-pro/ai-toolkit
```

将扩展添加到你的编辑器，并访问其方法：

```tsx
import { Editor } from '@tiptap/core'
import StarterKit from '@tiptap/starter-kit'
import { AiToolkit, getAiToolkit } from '@tiptap-pro/ai-toolkit'

const editor = new Editor({
  // 添加 AI 工具包扩展
  extensions: [AiToolkit],
})

// 获取 AI 工具包实例
const toolkit = getAiToolkit(editor)

// 调用 AI 工具包方法
toolkit.insertText('AI 生成的内容')
```

恭喜！你已成功设置 AI 工具包。现在你可以开始构建你的 AI 集成了。
