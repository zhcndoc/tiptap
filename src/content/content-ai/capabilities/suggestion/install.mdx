## 安装
AI 建议扩展已发布在 Tiptap 的私有 npm 注册表中。按照 [私有注册表指南](/guides/pro-extensions) 集成扩展。

一旦准备就绪，您可以像安装任何其他 Tiptap Pro 扩展一样进行安装。

```bash
npm install @tiptap-pro/extension-ai-suggestion@beta
```

## 在编辑器中导入扩展

将 `AiSuggestion` 添加到扩展列表中。

```ts
import { Editor } from '@tiptap/core'
import StarterKit from '@tiptap/starter-kit'
import AiSuggestion from '@tiptap-pro/extension-ai-suggestion'

// 初始化编辑器
const editor = new Editor({
  extensions: [
    StarterKit,
    AiSuggestion.configure({
      // 定义建议规则
      rules: [
        {
          id: '1',
          title: '拼写检查',
          prompt: '识别并纠正任何拼写错误',
          color: '#DC143C',
          backgroundColor: 'FFE6E6',
        },
      ],
      // 配置内容 AI 云（见下一部分）
      appId: 'APP_ID_HERE',
      token: 'TOKEN_HERE',
      // … 其他选项
    }),
    // … 更多扩展
  ],
  // 推荐：禁用拼写检查以避免冲突
  editorProps: {
    attributes: {
      spellcheck: false,
    },
  },
})
```

## 决定建议生成方式

下一步是配置生成将在编辑器中显示的建议的方式。

开始使用 AI 建议扩展的最简单方法是使用 Tiptap 内容 AI 云。我们的 API 将根据您的规则和内容生成建议。请按照此设置指南：[使用 Tiptap 内容 AI 云](/content-ai/capabilities/suggestion/use-with-content-ai-cloud)。

您也可以使用自己的后端和 LLM 生成校对建议。您可以在这里找到指南：[集成您的专用后端和 LLM](/content-ai/capabilities/suggestion/custom-llms)。

## 自定义用户界面

最后，您需要配置编辑器样式，以便在 UI 中显示建议。

默认情况下，AI 建议具有 CSS 类和颜色属性，以帮助您为其样式化。使用这些 CSS 样式在建议下方显示彩色下划线。

```css
.tiptap-ai-suggestion {
  border-bottom: 2px solid var(--tiptap-ai-suggestion-color);
  margin-bottom: -2px;
}

.tiptap-ai-suggestion--selected {
  background-color: var(--tiptap-ai-suggestion-background-color);
  transition: background-color 0.5s;
}
```

您可以通过提供自定义装饰进一步自定义建议的外观。要在选择建议时显示工具提示或弹出窗口，请 [遵循此指南](/content-ai/capabilities/suggestion/features/display-suggestions#show-a-popover-when-you-select-a-suggestion)。
