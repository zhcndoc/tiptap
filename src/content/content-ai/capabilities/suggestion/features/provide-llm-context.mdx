当你 [使用 Tiptap 内容 AI 云生成建议](/content-ai/capabilities/suggestion/use-with-content-ai-cloud) 时，以下信息将被发送到 LLM：

- 编辑器的内容。
- 扩展的 [规则](/content-ai/capabilities/suggestion/features/define-rules)。
- `context` 选项（如果提供）。

`context` 选项是一个可选的字符串属性，用于向 LLM 提供额外信息。此信息将被添加到提示中。它是全局的，因此将与所有规则一起使用。

## 配置上下文的初始值

你可以在扩展的配置中提供上下文的初始值。

```ts
AiSuggestion.configure({
  context: '语气应正式且专业。',
})
```

## 更新上下文

你可以随时使用 `setAiSuggestionContext` 命令更新上下文的值。如果你想根据用户输入或其他应用程序特定的逻辑更改上下文，这会很有用。

```ts
const newContext = '语气应非正式且友好。'
editor.commands.setAiSuggestionContext(newContext)
```

此命令不会自动重新加载建议。你需要调用 `loadAiSuggestions` 命令，以基于新的上下文更新建议。一种常见的模式是将两个命令连锁在一起。

```ts
editor.chain().setAiSuggestionContext(newContext).loadAiSuggestions().run()
```

要清除上下文，将其设置为 `null`。

```ts
editor.commands.setAiSuggestionContext(null)
```

## 读取当前上下文

你可以通过 `context` 存储属性访问当前上下文的值。

```ts
const currentContext = editor.extensionStorage.aiSuggestion.context
```
