因为您的文档可能包含重要信息，您可能不想对其所有内容生成建议。您可能想将某些内容标记为“已锁定”，并防止生成修改它的建议。

我们正在开发一个扩展来实现这一功能，尽量减少麻烦。如果您对即将推出的这一功能感兴趣，并认为您会在应用程序中使用它，请通过 humans@tiptap.dev 与我们联系。

您还可以通过过滤解析器生成的建议来防止某些内容被校对。实现您自定义的过滤函数，以删除应用于文档某些位置的建议，或者删除包含特定类型内容的建议。

```ts
AiSuggestion.configure({
  async resolver({ defaultResolver, ...options }) {
    // 加载建议
    const suggestions = await defaultResolver(options)
    // 移除修改已锁定内容的建议
    const filteredSuggestions = filterSuggestions(suggestions)
    return filteredSuggestions
  },
})

// 过滤函数，移除修改 10 到 20 位置内容的建议。
function filterSuggestions(suggestions) {
  return suggestions.filter(
    (suggestion) => suggestion.deleteRange.from > 20 || suggestion.deleteRange.to < 10,
  )
}
```
