开始使用 AI 建议扩展的最快方法是通过 Tiptap 内容 AI 云 API 生成建议。这是开箱即用的，您只需为扩展提供认证凭证。

此安装指南依赖于使用 [Tiptap 内容 AI 后端服务](https://cloud.tiptap.dev/v2/cloud/ai)，如果您想使用自己的自定义 LLM 进行配置，您需要遵循 [此指南](/content-ai/capabilities/suggestion/custom-llms)。

## 开始您的内容 AI 应用程序
1. **激活一个计划:** 开始一个 [免费试用或选择订阅](https://cloud.tiptap.dev/v2/billing)。
2. **配置您的 AI 应用。** 将您的 OpenAI API 密钥添加到您的 [内容 AI 应用](https://cloud.tiptap.dev/v2/cloud/ai)。
3. **设置认证**：生成一个 JWT（HS256 算法），使用我们提供的密钥对扩展进行身份验证以连接我们的服务。[获取您的 JWT 密钥。](https://cloud.tiptap.dev/v2/cloud/ai)

## 将您的认证凭证添加到扩展中

导入扩展并使用您的 Tiptap 内容 AI 应用 id 和令牌进行配置。

```tsx
import { Editor } from '@tiptap/core'
import StarterKit from '@tiptap/starter-kit'
import AiSuggestion from '@tiptap-pro/extension-ai-suggestion'

// 初始化编辑器
const editor = new Editor({
  extensions: [
    StarterKit,
    AiSuggestion.configure({
      // 您的 Tiptap 内容 AI 应用 id
      appId: 'APP_ID_HERE',
      // 这需要是您生成的 JWT，绝对不能是 OpenAI API 密钥！
      token: 'TOKEN_HERE',
      // 其他配置选项...
    }),
  ],
})
```

现在，当编辑器加载时，建议将从 Tiptap 内容 AI API 中获取。
