Tiptap 内容 AI 中的 `aiImagePrompt` 命令使您可以直接在编辑器中生成图像。您可以使用不同的 OpenAI 模型并自定义生成图像的风格。

要查看如何使用此命令，请查看 [类似 Notion 的模板](https://templates.tiptap.dev/)。

## 集成 aiImagePrompt

根据您的提示和所需的风格生成图像。

确保在您的编辑器实例中加载图像扩展（`'@tiptap/extension-image'`）。

```js
editor.chain().focus().aiImagePrompt(options: ImageOptions).run()
```

## 图像命令选项

通过这些设置，您可以控制图像的生成方式：

| 设置       | 类型                                                                                              | 默认值           | 定义                     |
| ---------- | ------------------------------------------------------------------------------------------------- | ---------------- | ------------------------ |
| modelName  | `dall-e-2`, `dall-e-3`, `null`                                                                    | `dall-e-3`       | 在 OpenAI 中使用的模型   |
| style      | `photorealistic`, `digital_art`, `comic_book`, `neon_punk`, `isometric`, `line_art`, `3d_model` | `photorealistic` | 定义图像风格             |
| size       | `256x256`, `512x512`, `1024x1024`                                                                 | `null`           |
