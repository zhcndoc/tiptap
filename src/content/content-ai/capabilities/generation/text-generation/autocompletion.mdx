启用自动完成时，系统会使用您已经编写的文本的一部分来生成建议。

要触发自动完成，您只需在编写时按 `Tab` 键。再次按 `Tab` 以接受建议的完成。

如果您想看到实际效果，在此演示中的任何文本行后按 `Tab` 键。

## 配置自动完成

在集成内容 AI 扩展时，您可以指定想要启用自动完成、
自动完成功能的行为以及该考虑什么上下文。

| 设置                     | 类型      | 默认值                                                                                     | 定义                                                                                                                                                                                                                                                     |
| ------------------------ | --------- | ------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `autocompletion`         | `boolean` | `false`                                                                                     | 启用自动完成功能。在编写文本时，只需按 **Tab** 以触发自动完成，再按一次 **Tab** 接受完成。我们将使用您已写文本的一部分来构建 OpenAI 的提示。                                                                                                        |
| `autocompletionOptions`  | `object`  | `{ trigger: 'Tab', accept: 'Tab', debounce: 0, inputLength: 4000, modelName: 'gpt-3.5-turbo' }` | 定义生成自动完成建议所使用的触发器和文本长度。如果未明确设置，接受默认为触发器。触发后请求的去抖动时间（毫秒）。您还可以选择运行自动完成任务的 OpenAI 模型。                                                                                     |

要查看所有其他内容 AI 扩展配置选项，请前往 [安装](/content-ai/capabilities/generation/install#configure-the-extension) 页面。

## 启用自动完成

```js
import { Editor } from '@tiptap/core'
import StarterKit from '@tiptap/starter-kit'
import Ai from '@tiptap-pro/extension-ai'

const editor = new Editor{
  extensions: [
    StarterKit,
    Ai.configure({
      appId: 'APP_ID_HERE',
      token: 'TOKEN_HERE',
      autocompletion: true,
      // … 其他选项（见上文）
    }),
    // … 更多扩展
  ],
})
```
