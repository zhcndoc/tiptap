通过 `context` 选项，你可以向 AI 模型提供额外信息，帮助其生成更相关的内容。当你希望生成更符合对话上下文的内容时，这非常有用。

<CodeDemo isPro path="/Extensions/AiContext" />

## context 选项

上下文可采用两种格式，`text` 和 `url`。`text` 格式用于向 AI 模型提供额外的文本上下文，而 `url` 格式用于提供 URL 上下文。

<Callout title="URL 要求" variant="warning">
    所提供的 URL 必须是有效且可被 Tiptap 内容 AI 访问的。这在你希望提供引用第三方来源的额外上下文时非常有用。
</Callout>

```ts
// 以富文本流式传输响应到编辑器
editor
  .chain()
  .aiTextPrompt({
    text: 'Alice 是做什么工作的？',
    stream: true,
    format: 'rich-text',
    context: [
      { type: 'text', text: 'John 和 Alice 是软件工程师。' },
      { type: 'url', url: 'https://company.example.com/team' },
    ],
  })
  .run()
```
