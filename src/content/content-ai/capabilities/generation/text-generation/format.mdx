通过 `format: 'rich-text'` 选项，您可以生成应用格式化内容的 AI 响应。这在您想生成包含粗体、斜体、链接等丰富文本格式的内容时特别有用。

```ts
// 将响应作为丰富文本流入编辑器
editor
  .chain()
  .aiTextPrompt({
    text: '写出一份流行编程语言的列表',
    stream: true,
    format: 'rich-text',
  })
  .run()
```
