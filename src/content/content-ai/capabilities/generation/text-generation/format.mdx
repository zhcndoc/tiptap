通过 `format: 'rich-text'` 选项，你可以生成带有内容格式的 AI 响应。这在你需要生成包含加粗、斜体、链接等富文本格式的内容时特别有用。

```ts
// 将响应以富文本格式流式传输到编辑器
editor
  .chain()
  .aiTextPrompt({
    text: '写一份流行编程语言列表',
    stream: true,
    format: 'rich-text',
  })
  .run()
```
