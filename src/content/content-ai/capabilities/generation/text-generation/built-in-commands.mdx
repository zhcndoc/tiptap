Tiptap 编辑器的 AI 生成扩展包含一组预配置的命令，您可以将其集成到您的富文本编辑器中。这些命令允许您调整文本语气、完成文本、生成图像等，从而增强您编辑器的功能。

要查看这些命令的用法，请查看[概述](/content-ai/capabilities/generation/overview)页面上的示例。

| ------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |

| [`aiBloggify(options: TextOptions)`](#most-text-commands)                                                                       | 将文本重写为博客格式                                                                                                             |
| [`aiComplete(options: TextOptions)`](#most-text-commands)                                                                       | 完成所选文本                                                                                                                       |
| [`aiDeEmojify(options: TextOptions)`](#most-text-commands)                                                                      | 从所选文本中删除表情符号                                                                                                          |
| [`aiEmojify(options: TextOptions)`](#most-text-commands)                                                                        | 向您的文本添加表情符号 ✨                                                                                                         |
| [`aiExtend(options: TextOptions)`](#most-text-commands)                                                                         | 扩展您的文本                                                                                                                       |
| [`aiFixSpellingAndGrammar(options: TextOptions)`](#most-text-commands)                                                          | 修正拼写和语法                                                                                                                    |
| [`aiKeypoints(options: TextOptions)`](#most-text-commands)                                                                      | 将您的文本总结为关键点列表                                                                                                        |
| [`aiRephrase(options: TextOptions)`](#most-text-commands)                                                                       | 重新措词所选文本                                                                                                                  |
| [`aiRestructure(options: TextOptions)`](#most-text-commands)                                                                    | 重新构建所选文本以使用富文本格式                                                                                                    |
| [`aiShorten(options: TextOptions)`](#most-text-commands)                                                                        | 缩短所选文本                                                                                                                       |
| [`aiSimplify(options: TextOptions)`](#most-text-commands)                                                                       | 用简化的词语重新措词您的文本                                                                                                        |
| [`aiSummarize(options: TextOptions)`](#most-text-commands)                                                                      | 总结您的文本                                                                                                                       |
| [`aiTextPrompt(options: TextOptions)`](#most-text-commands)                                                                     | 运行您的自定义提示                                                                                                                |

### 大多数文本命令

大多数文本命令接受相同的选项，使用方法相似。以下示例演示如何使用 `aiBloggify` 命令：

```js
editor.chain().focus().aiBloggify(options: TextOptions)
```

### aiAdjustTone

```js
// 语气: 'default' | 'academic' | 'business' | 'casual' | 'childfriendly' | 'confident' | 'conversational' | 'creative' | 'emotional' | 'excited' | 'formal' | 'friendly' | 'funny' | 'humorous' | 'informative' | 'inspirational' | 'memeify' | 'narrative' | 'objective' | 'persuasive' | 'poetic' | string
editor.chain().focus().aiAdjustTone(tone: Tone, options: TextOptions).run()
```

### aiTranslate

将所选文本内容翻译成给定的输出语言。

它接受两位字母的 ISO 639-1 语言代码。

```js
// 语言: 'en' | 'de' | 'nl' | ...
editor.chain().focus().aiTranslate(language: Language, options: TextOptions).run()
```

## 文本命令选项

在每个支持 `TextOptions` 的命令上，您可以指定以下选项：

| 设置              | 类型                                                                   | 默认值         | 定义                                                                                                                                                                                                                  |
| ---------------- | ---------------------------------------------------------------------- | --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `modelName`      | 查看 [支持的文本模型](#supported-text-models)                         | `gpt-3.5-turbo` | 在 OpenAI 中使用的模型                                                                                                                                                                                              |
| `format`         | `'rich-text' \| 'plain-text'`                                         | `plain-text`    | 确定模型应如何响应的[格式](/content-ai/capabilities/generation/text-generation/format)                                                                                                                           |
| `stream`         | `boolean`                                                              | `false`         | 命令是否应该将字符流式传输到编辑器？这类似于 ChatGPT 中的 **打字机** 行为。**这需要最新的扩展版本！**                                                                                                          |
| `insertAt`       | `undefined \| number \| {from: number; to: number}`                   | `undefined`     | 如果为 `undefined`，则响应将插入在当前位置。如果为数字，响应将插入该位置。如果为对象，则将替换该范围。                                                                                                         |
| `language`       | `string` (e.g. `en`, `de`)                                            | `null`          | 尽管我们尽力提示 OpenAI 以输入语言响应，有时最好是您自己定义它。                                                                                                                                                    |
| `tone`           | `string`                                                               | `null`          | 响应应转换为的[语气](#ai-adjust-tone)                                                                                                                                            |
| `textLength`     | `number`                                                               | `undefined`     | AI 应该响应的 `textLengthUnit` 的数量（例如，"3 段" 中的 `3`）                                                                                                                                                     |
| `textLengthUnit` | `'paragraphs' \| 'words' \| 'characters'`                            | `undefined`     | AI 应该响应的 `textLength` 的单位（例如，"3 段" 中的 `paragraphs`）                                                                                                                                           |
| `collapseToEnd`  | `boolean`                                                              | `true`          | 操作完成后，光标是否应设置到末尾或插入是否应被选中。                                                                                                                                                             |
| `context`        | `Array<{ type: 'text', text: string} \| { type: 'url', url: string }>` | `[]`            | [上下文](/content-ai/capabilities/generation/text-generation/provide-context) 以便为生成响应提供模型。                                                                                                           |
| `text`           | `string`                                                               | `undefined`     | 一个可选的消息以让模型响应                                                                                                                                                                                         |
| `insert`         | `boolean`                                                              | `true`          | 允许禁用直接将内容插入编辑器                                                                                                                                                                                        |
| `append`         | `boolean`                                                              | `true`          | 如果为 `true`，响应将追加到当前选择的末尾。如果为 `false`，响应将替换当前选择。                                                                                                                                 |

_不幸的是，**tone** 和 **language** 的组合有时会导致响应不是所需语言。_

## 选择合适的模型

配置 Tiptap AI 扩展时，请考虑您应用的具体需求：

- **对于成本效益的运营：** 如果主要关注预算，且任务不需要最先进的能力，请选择 GPT-3 或 DALL-E 2。
- **对于高级需求：** 当您的应用需要最高级别的语言理解或图像生成质量，且预算不太受限制时，请选择 GPT-4o 或 DALL-E 3。

Tiptap AI 扩展的灵活配置使您能够根据您的具体要求和预算考虑调整 AI 集成。
