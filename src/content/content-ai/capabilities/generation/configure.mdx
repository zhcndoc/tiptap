Tiptap 的内容 AI 扩展接受不同的设置，以配置扩展及其命令的全局行为。

| 设置                     | 类型                                                                                                                            | 默认值                                                                                          | 定义                                                                                                                                                                                                                                                  |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `appId`                 | `string`                                                                                                                      | `''`                                                                                           | 您的应用程序 ID，可以通过 [访问 Tiptap Cloud 中的 AI 设置](https://collab.tiptap.dev/ai-settings) 获取                                                                                                                                        |
| `token`                 | `string`                                                                                                                      | `''`                                                                                           | 为了与我们的 AI 后端服务进行身份验证，您需要使用您的 JWT 秘密生成一个 JWT（使用 HS256），该密钥您也可以在 [团队的 AI 设置页面](https://cloud.tiptap.dev/ai-settings) 获取。                                                                 |
| `autocompletion`        | `boolean`                                                                                                                     | `false`                                                                                        | 启用自动补全功能。编写文本时，只需按 **Tab** 触发自动补全，再按一次 **Tab** 接受补全。我们将使用您已写的文本的一部分为 OpenAI 构建提示。                                                                                                         |
| `autocompletionOptions` | `object`                                                                                                                      | `{ trigger: 'Tab', accept: 'Tab', debounce: 0, inputLength: 4000, modelName: 'gpt-3.5-turbo' }` | 定义生成自动补全建议所用的触发器和文本长度。如果没有明确设置，接受将默认为触发器。按下触发器后，以毫秒为单位的去抖动请求。您还可以选择用于执行自动补全任务的 OpenAI 模型。                                                                           |
| `onLoading`             | `(context: { editor: Editor, action: TextAction \| 'image', isStreaming: boolean }) => void`                                  | `undefined`                                                                                    | 当 AI 开始生成响应时的回调。                                                                                                                                                                                                                            |
| `onSuccess`             | `(context: { editor: Editor, action: TextAction \| 'image', isStreaming: boolean, response: string }) => void`                | `undefined`                                                                                    | 当 AI 成功完成生成响应时的回调。                                                                                                                                                                                                                      |
| `onChunk`               | `(context: { editor: Editor, action: TextAction \| 'image', isStreaming: boolean, response: string, chunk: string }) => void` | `undefined`                                                                                    | 当生成的响应的一部分生成时的回调。chunk 是部分响应的新字符串，包含到目前为止生成的响应（包括该部分）。                                                                                                                                                                       |
| `onError`               | `(error: Error, context: { editor: Editor, action: TextAction \| 'image', isStreaming: boolean }) => void`                    | `undefined`                                                                                    | 当生成响应时发生错误的回调。                                                                                                                                                                                                                          |
| `showDecorations`       | `boolean`                                                                                                                     | `true`                                                                                         | 如果为 `false`，则在流模式下不会尝试装饰 AI 建议。这减少了您对建议进行样式设置的能力，但可以解决撤销历史中的问题。                                                                                                                                                          |
