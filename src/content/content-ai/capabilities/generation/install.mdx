## 开始您的内容 AI 应用
1. **激活计划：** 开始一个 [免费试用或选择订阅](https://cloud.tiptap.dev/v2/billing)。
2. **配置您的 AI 应用。** 将您的 OpenAI API 密钥添加到您的 [内容 AI 应用](https://cloud.tiptap.dev/v2/cloud/ai)。
3. **设置认证**：使用我们提供的密钥生成 JWT（HS256 算法），以便将扩展与我们的服务进行身份验证。 [获取您的 JWT 密钥。](https://cloud.tiptap.dev/v2/cloud/ai)

## 访问专业注册表
AI 生成扩展发布在 Tiptap 的私有 npm 注册表中。通过遵循 [私有注册表指南](/guides/pro-extensions) 来集成扩展。如果您已经对 Tiptap 账户进行了身份验证，可以直接跳转到 [#安装](#install)。

## 安装

```bash
npm install @tiptap-pro/extension-ai@beta
```

将其集成到您的编辑器实例中的方式与其他 Tiptap 扩展类似。以下是一个示例：

```js
import { Editor } from '@tiptap/core'
import StarterKit from '@tiptap/starter-kit'
import Ai from '@tiptap-pro/extension-ai'

const editor = new Editor({
  extensions: [
    StarterKit,
    Ai.configure({
      // 您的 Tiptap 内容 AI 应用 ID
      appId: 'APP_ID_HERE',
      // 这必须是您生成的 JWT，绝对不要是 OpenAI API 密钥！
      token: 'TOKEN_HERE',
      autocompletion: true,
      // … 其他选项（见下文）
    }),
    // … 更多扩展
  ],
})
```

到此，您可以在 Tiptap 编辑器中使用 OpenAI。可以查看 [配置选项](/content-ai/capabilities/generation/configure) 来定制您的体验。
