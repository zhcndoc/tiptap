### 以效率为核心构建

Server AI Toolkit 适用于对成本敏感的使用场景。可用的工具，如 `tiptapEdit` 和 `proofread`，采用高效格式，使 AI 能够快速应用编辑并保持较低的 token 消耗。

[Tiptap Shorthand](/content-ai/capabilities/server-ai-toolkit/advanced-guides/tiptap-shorthand)（alpha）是一种用于编码 Tiptap 文档的 token 高效格式。它在不牺牲准确性的前提下，可将 AI token 成本降低高达 80%。

它利用专门的编辑格式 Tiptap Edit，让 AI 进行小而精准的编辑，无需替换整个文档。

Server AI Toolkit 可以作为云服务使用，也可以在本地部署，完全运行在你自己的基础设施上。
