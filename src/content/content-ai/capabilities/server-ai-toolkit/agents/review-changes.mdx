当 Server AI Toolkit 在服务器端编辑文档后，你可以在客户端显示一个审核界面，让用户在更改应用之前接受或拒绝这些更改。

本指南展示如何在审核模式下使用客户端 AI Toolkit 的 `setSuggestionsFromDiff` 方法，以显示原始文档与编辑版本之间的差异。

## 工作原理

1. 开发者在 AI 进行任何更改之前，将当前文档存储在 `originalDoc` 变量中。
2. AI 修改文档并返回更新后的版本。
3. 在客户端，开发者调用 `setSuggestionsFromDiff`，并将 `originalDoc` 作为要比较的文档传入。
4. AI Toolkit 比较原始文档与当前版本，并将差异显示为审核建议。
5. 用户可以接受或拒绝单个更改，也可以一次性接受/拒绝所有更改。

## 显示差异

从 Server AI Toolkit 接收到更新后的文档后，在审核模式下使用 `setSuggestionsFromDiff` 来显示更改：

```ts
import { getAiToolkit } from '@tiptap-pro/ai-toolkit'

// 从编辑器获取 AI Toolkit 实例
const toolkit = getAiToolkit(editor)

// 在 AI 进行更改之前存储文档。
const originalDoc = editor.state.doc

// 等待 AI 进行工具调用并编辑文档。

// 文档编辑后，显示旧文档与新文档之间的差异
toolkit.setSuggestionsFromDiff({
  reviewOptions: {
    // 在审核模式下显示建议
    mode: 'review',
  },
  doc: originalDoc,
})
```

编辑器现在将显示原始文档与服务器编辑版本之间的差异，删除内容以红色突出显示，插入内容以绿色突出显示。

## 定制差异样式

添加 CSS 样式，将建议内容以红/绿差异突出显示：

```css
/* 将插入的文本突出显示为绿色 */
.tiptap-ai-suggestion,
.tiptap-ai-suggestion > * {
  background-color: oklch(87.1% 0.15 154.449);
}

/* 更改组的背景色较浅（子更改带有更强烈的突出显示） */
.tiptap-ai-suggestion.tiptap-ai-suggestion--change-group,
.tiptap-ai-suggestion.tiptap-ai-suggestion--change-group > *:not(.tiptap-ai-suggestion-sub-change) {
  background-color: oklch(0.962 0.044 156.743);
}

/* 突出显示内联组内的子更改 */
.tiptap-ai-suggestion-sub-change {
  background-color: oklch(87.1% 0.15 154.449);
}

/* 将删除的文本突出显示为红色 */
.tiptap-ai-suggestion-diff,
.tiptap-ai-suggestion-diff > * {
  background-color: oklch(80.8% 0.114 19.571);
  color: oklch(0.396 0.141 25.723);
}

/* 差异更改组的背景色较浅（子更改带有更强烈的突出显示） */
.tiptap-ai-suggestion-diff.tiptap-ai-suggestion-diff--change-group,
.tiptap-ai-suggestion-diff.tiptap-ai-suggestion-diff--change-group
  > *:not(.tiptap-ai-suggestion-diff-sub-change) {
  background-color: oklch(0.936 0.032 17.717);
}

/* 突出显示替换差异组件内的子更改 */
.tiptap-ai-suggestion-diff-sub-change {
  background-color: oklch(80.8% 0.114 19.571);
}

/* 正确渲染表格行删除 */
.tiptap-ai-suggestion-diff:has(tr) {
  display: contents;
}

.tiptap-ai-suggestion-diff:has(tr) td,
.tiptap-ai-suggestion-diff:has(tr) th {
  background-color: oklch(80.8% 0.114 19.571);
}
```

## 接受或拒绝更改

差异显示后，用户可以通过建议方法接受或拒绝更改：

```ts
const toolkit = getAiToolkit(editor)

// 接受所有更改
toolkit.acceptAllSuggestions()

// 拒绝所有更改
toolkit.rejectAllSuggestions()

// 或处理单个建议
const suggestions = toolkit.getSuggestions()
toolkit.acceptSuggestion(suggestions[0].id)
toolkit.rejectSuggestion(suggestions[1].id)
```
