## 可用工具

Server AI Toolkit 提供了 [工具定义](/content-ai/capabilities/server-ai-toolkit/api-reference/rest-api)，您的 AI 代理可以调用这些工具与服务器上的文档进行交互。这些工具能够高效地读取内容并进行编辑。

当您调用 Server AI Toolkit API 端点时，AI 模型可以生成工具调用，然后通过 `/v4/ai/toolkit/execute-tool` 端点来执行。这使得 AI 能够根据用户请求读取、编辑和操作文档。
