你可以查看[我们的自定义扩展指南](/hocuspocus/guides/custom-extensions)了解如何创建你自己的扩展。

## 目录

我们已经创建了一些非常有用的扩展，你绝对应该查看：

| 扩展                                        | 描述                                                                          |
|----------------------------------------------|--------------------------------------------------------------------------------|
| [数据库](/hocuspocus/server/extensions/database)       | 通用数据库驱动，易于调整以适配任何数据库。                                     |
| [日志记录器](/hocuspocus/server/extensions/logger)     | 为 Hocuspocus 添加日志记录功能。                                               |
| [Redis](/hocuspocus/server/extensions/redis)           | 使用 Redis 实现 Hocuspocus 的横向扩展。                                        |
| [SQLite](/hocuspocus/server/extensions/sqlite)         | 将文档持久化到 SQLite。                                                       |
| [限流](/hocuspocus/server/extensions/throttle)         | 按 IP 限制连接速率。                                                          |
| [Webhook](/hocuspocus/server/extensions/webhook)       | 通过 webhook 将文档变更发送到你的 API。                                       |
| [S3](/hocuspocus/server/extensions/s3)                 | 将文档持久化到 Amazon S3 或任何兼容 S3 的存储服务。                           |
