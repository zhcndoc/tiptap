## 介绍
你的 Hocuspocus 服务器已经运行了吗？是时候试用一下 Hocuspocus Provider 了！从 npm 安装它，几行代码即可开始使用。

## 安装 Provider

```bash
npm install yjs @hocuspocus/provider
```

## 用法
使用时，创建一个新的 Y.js 文档（如果你还没有的话），并将其附加到 Hocuspocus Provider 上。

```js
import * as Y from 'yjs'
import { HocuspocusProvider } from '@hocuspocus/provider'

const ydoc = new Y.Doc()

const provider = new HocuspocusProvider({
  url: 'ws://127.0.0.1:1234',
  name: 'example-document',
  document: ydoc,
})
```

完成！你现在已经将一个 Y.js 文档连接到 Hocuspocus 服务器。对 Y.js 文档的更改将同步到所有其他已连接的用户。

如果你正在构建一个 React 应用，请看看 [React bindings](/hocuspocus/provider/react) —— 它们会将 provider 封装在组件和 hooks 中，这样你就不必手动管理生命周期。
