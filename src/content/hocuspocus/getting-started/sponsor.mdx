## 介绍

为了提供顶级的开发者体验和用户体验，我们在 Hocuspocus 上投入了数百小时的无偿工作。您的资金支持帮助我们使这项工作变得更加经济可持续。这使我们能够提供有用的支持，维护我们所有的包，保持所有内容的最新状态，并为 Hocuspocus 开发新功能和扩展。

回馈开源社区，欢迎 [在 GitHub 上赞助我们](https://github.com/sponsors/ueberdosis)！ ♥

## 常见问题

### 我不能使用 GitHub。怎样才能支持你们？

如果您是公司，不想使用 GitHub，或者没有信用卡，或者想要正式的发票，请直接通过 [humans@tiptap.dev](mailto:humans@tiptap.dev) 与我们联系。

我们有一个 [OpenCollective](https://opencollective.com/tiptap)，允许您通过转账、PayPal 或信用卡汇款。对美国公司来说，捐款是可抵税的。

### 我想咨询。你们的收费标准是多少？

如果您遇到问题、有疑问、想讨论某些事情或其它任何情况，请 [使用 GitHub 问题区](https://github.com/ueberdosis/hocuspocus/issues)，以便所有社区成员都能访问。其他任何情况，请联系 [humans@tiptap.dev](mailto:humans@tiptap.dev)。我们可以接受有限数量的定制开发和咨询合同。
