一个用于向选定文本添加评论或注释的按钮。

## 安装

```bash
npx @tiptap/cli add comment-button
```

## 用法

```tsx
import { CommentButton } from '@/components/tiptap-ui/comment-button'
import { useEditor, EditorContent } from '@tiptap/react'

export default function MyEditor() {
  const editor = useEditor({
    // 编辑器配置
  })

  return (
    <div>
      <CommentButton
        editor={editor}
        hideWhenUnavailable={false}
        onClick={() => console.log('添加评论')}
      />
      <EditorContent editor={editor} />
    </div>
  )
}
```

## 属性

| 名称                       | 类型           | 默认值  | 描述                                               |
| -------------------------- | -------------- | ------- | -------------------------------------------------- |
| editor                     | Editor \| null | null    | Tiptap 编辑器实例                                 |
| hideWhenUnavailable        | boolean        | false   | 当不可用时按钮是否应隐藏                          |
| showOnResolvedBlockThreads | boolean        | true    | 控制已解决块线程的可见性                          |
