`Comments` 扩展允许用户在编辑器中创建线程和评论。线程可用于讨论文档的特定部分，而单独的评论可以添加到特定部分。

评论可以通过 [Comments REST API](/comments/integrate/rest-api) 访问和操作，也可以通过 [webhooks](/comments/integrate/webhook) 接收，从而实现通知系统的创建以及从编辑器外部添加评论的功能。

    </RequirementItem>

## 评论功能

- 添加行内、文档或侧边栏评论
- 对文本、节点、自定义节点或节点选择进行评论
- 评论中支持富文本（例如，加粗、表情）
- 解决、编辑或删除评论
- 离线评论支持
- 处理重叠评论
- 在评论中直接提及用户
- 当用户被提及时，通过 Webhook 触发自定义通知服务
- 使用评论 API 程序化管理评论
