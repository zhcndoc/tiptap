## Import Markdown

`POST /v2/convert/import/markdown`

The `/v2/convert/import/markdown` endpoint converts Markdown files (`.md` or `.markdown`) to Tiptap JSON format. Upload files using `multipart/form-data`.

### Example (cURL)

```bash
curl -X POST "https://api.tiptap.dev/v2/convert/import/markdown" \
    -H "Authorization: Bearer YOUR_TOKEN" \
    -H "X-App-Id: YOUR_APP_ID" \
    -F "file=@/path/to/file.md"
```

<Callout title="Subscription Required" variant="warning">
    This endpoint requires an active Tiptap subscription. See our [pricing page](https://tiptap.dev/pricing) for more details.
</Callout>

### Required Headers

| Name             | Description                                                                                                   |
| ---------------- | ------------------------------------------------------------------------------------------------------------- |
| `Authorization`  | JWT token to authenticate the request, e.g., `Bearer your-jwt-token`                                          |
| `X-App-Id`       | Convert App ID from the Convert settings page: [https://cloud.tiptap.dev/v2/cloud/convert](https://cloud.tiptap.dev/v2/cloud/convert) |

### Request Body

| Name       | Type      | Required | Description                                     |
| ---------- | --------- | -------- | ---------------------------------------------- |
| `file`     | `File`    | Yes      | Markdown file to convert (`.md` or `.markdown`) |
| `verbose`  | `string`  | No       | Log verbosity level (bitmask: 1=log, 2=warn, 4=error) |

### Response (200 OK)

```json
{
  "data": {
    "content": {
      "type": "doc",
      "content": [
        {
          "type": "paragraph",
          "content": [
            {
              "type": "text",
              "text": "Hello from Markdown!"
            }
          ]
        }
      ]
    }
  },
  "logs": []
}
```

### Error Responses

| Status Code | Error Code                        | Description                          |
| ----------- | -------------------------------- | ---------------------------------- |
| 400         | `NO_FILE_PROVIDED`               | No file provided                   |
| 400         | `INVALID_FILE_TYPE`              | File is not `.md` or `.markdown`  |
| 422         | `FAILED_TO_GET_FILE_ARRAY_BUFFER`| Failed to read file buffer         |
| 500         | `FAILED_TO_CONVERT_MARKDOWN_TO_TIPTAP_JSON` | Markdown to JSON conversion failed |
