### 段落样式对象

`paragraphStyle` 对象接受以下属性：

| 属性          | 类型    | 描述                                                               |

## 元素覆盖

元素覆盖与 `styleOverrides` 是分开的。虽然 `styleOverrides` 定义了 Word 通过其样式系统应用的**命名段落样式**（标题 1、正文等），但**元素覆盖**在转换过程中直接将基础默认值应用于每个段落、文本运行、表格、表格单元格或图像。

这让您无需定义命名样式即可对每种元素的原始 docx 属性进行细粒度控制。

> **浅层展开**：所有覆盖对象都使用浅层展开，而非深层合并。在覆盖如 `spacing` 或 `transformation` 等嵌套属性时，请提供完整的嵌套对象，以避免出现未定义字段。

### `paragraphOverrides`

接受除 `children` 外的任何 `IParagraphOptions` 属性。这些值首先作为基础默认值进行展开 —— 每个节点计算出的值（对齐、间距、标题级别、列表编号）会覆盖它们。

应用于：标准段落、标题、块引用、无序列表、有序列表和列表项。

```ts
ExportDocx.configure({
  onCompleteExport: (result) => { /* ... */ },
  paragraphOverrides: {
    spacing: { after: 200, before: 100 },
    alignment: AlignmentType.LEFT,
  },
})
```

### `textRunOverrides`

接受除 `text` 外的任何 `IRunOptions` 属性。这些值首先作为基础默认值进行展开 —— 每个标记的格式（粗体、斜体、下划线、字体、字号、颜色、高亮等）会覆盖它们。

可用于为整个导出的文档设置默认字体或字号。

```ts
ExportDocx.configure({
  onCompleteExport: (result) => { /* ... */ },
  textRunOverrides: {
    font: 'Arial',
    size: 24, // 单位为半点数的 12pt
  },
})
```

### `tableOverrides`

接受除 `rows` 外的任何 `ITableOptions` 属性。这些值最后进行展开 —— 用户提供的表格级属性（边框、边距、布局）优先于计算出的默认值。

```ts
ExportDocx.configure({
  onCompleteExport: (result) => { /* ... */ },
  tableOverrides: {
    width: { size: 100, type: WidthType.PERCENTAGE },
    layout: TableLayoutType.FIXED,
  },
})
```

### `tableCellOverrides`

接受除 `children` 外的任何 `ITableCellOptions` 属性。这些值首先作为基础默认值进行展开 —— 每个单元格计算出的值（宽度、跨列、宽度类型）会覆盖它们。

可用于为所有表格单元格应用一致的底纹、垂直对齐或边框样式。

```ts
ExportDocx.configure({
  onCompleteExport: (result) => { /* ... */ },
  tableCellOverrides: {
    shading: { fill: 'F0F0F0', type: ShadingType.SOLID },
    verticalAlign: VerticalAlign.CENTER,
  },
})
```

### `imageOverrides`

接受除 `data`、`type` 和 `fallback` 外的任何 `IImageOptions` 属性。这些值最后进行展开 —— 用户提供的值（例如自定义 `transformation` 尺寸）优先于从图像缓冲区检测到的固有尺寸。无论覆盖如何，SVG 回退数据始终被保留。

```ts
ExportDocx.configure({
  onCompleteExport: (result) => { /* ... */ },
  imageOverrides: {
    transformation: { width: 400, height: 300 },
    floating: {
      horizontalPosition: { offset: 0 },
      verticalPosition: { offset: 0 },
    },
  },
})
```

您可以在单个配置中组合所有元素覆盖：

```ts

    font: 'Arial',
    size: 24, // 单位为半点数的 12pt
  },
  tableCellOverrides: {
    shading: { fill: 'F0F0F0', type: ShadingType.SOLID },
  },
  imageOverrides: {
    transformation: { width: 400, height: 300 },
  },
})
```
