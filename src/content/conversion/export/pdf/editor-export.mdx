使用 Tiptap 的 `@tiptap-pro/extension-export-pdf` 可以将编辑器内容导出为 `.pdf` 文件。此扩展通过将文档发送至 Tiptap 转换服务，实现 PDF 导出功能并集成到编辑器中。

如果您更愿意在自己端处理转换，也可以使用[REST API](/conversion/export/pdf/rest-api)。

## 安装 PDF 导出扩展

转换扩展发布在 Tiptap 的私有 npm 注册表中。请按[私有注册表指南](/guides/pro-extensions)集成扩展。

完成后即可安装并导入**导出 PDF**扩展包。

```bash
npm i @tiptap-pro/extension-export-pdf
```

```js
import { ExportPdf } from '@tiptap-pro/extension-export-pdf'
```

## 配置扩展

`ExportPdf` 扩展可通过 `configure` 方法传入一个带有以下属性的 `ExportPdfOptions`（对象）进行配置：

```ts
import { ExportPdf } from '@tiptap-pro/extension-export-pdf'

const editor = new Editor({
  extensions: [
    // 其他扩展 ...
    ExportPdf.configure({
      endpoint: 'https://api.tiptap.dev/v2/convert',
      token: 'YOUR_TOKEN',
      appId: 'YOUR_APP_ID',
      styleOverrides: {},
      headers: {},
      footers: {},
      pageSize: { width: '21cm', height: '29.7cm' },
      pageMargins: { top: '1cm', bottom: '1cm', left: '1cm', right: '1cm' },
      customNodes: [],
      onCompleteExport: (result) => {
        // 处理导出的 Blob
      },
    }),
    // 其他扩展 ...
  ],
})
```

| Parameter        | Type                     | Description                                    | Default                               |
| ---------------- | ------------------------ | ---------------------------------------------- | ------------------------------------- |
| `endpoint`       | `string`                 | 用于转换的 Tiptap API 端点                     | `'https://api.tiptap.dev/v2/convert'` |
| `token`          | `string`                 | 用于认证的 Tiptap JWT                           | `''`                                  |
| `appId`          | `string`                 | 用于认证的 Tiptap 应用 ID                        | `''`                                  |
| `styleOverrides` | `Record<string, string>` | 应用于导出的样式覆盖                            | `undefined`                           |
| `headers`        | `HeaderConfig`           | 导出文档的页眉配置                              | `undefined`                           |
| `footers`        | `FooterConfig`           | 导出文档的页脚配置                              | `undefined`                           |
| `pageSize`       | `PageSize`               | 页面大小配置                                   | `undefined`                           |
| `pageMargins`    | `PageMargins`            | 页面边距配置                                   | `undefined`                           |
| `customNodes`    | `CustomNodeDefinition[]` | 用于基于 DOCX 转换的自定义节点定义（需要 `@tiptap-pro/extension-export-docx`） | `undefined` |
| `customFonts`    | `CustomFont[]`           | 为 PDF 导出提供的自定义字体文件（仅限本地部署）                 | `undefined`             |
| `onCompleteExport` | `(result: Blob) => void` | 接收导出文件 Blob 的回调函数                     | 如果未提供则抛出错误                    |

### HeaderConfig

| 属性                 | 类型      | 描述                                                                                                 |
| -------------------- | --------- | ---------------------------------------------------------------------------------------------------- |

| `first`              | `string`  | 首页的页眉，仅当 `differentFirstPage` 设置为 `true` 时生效。可接受纯文本字符串或字符串化的 Tiptap JSONContent（用于富格式）。   |
| `even`               | `string`  | 偶数页的页眉，当启用 `evenAndOddHeaders` 选项时生效。可接受纯文本字符串或字符串化的 Tiptap JSONContent（用于富格式）。         |

## 导出 PDF 文件

安装该扩展后，可以使用 `exportPdf` 命令将编辑器内容导出为 `.pdf`。

```ts
/**
 * 通过 Tiptap 转换服务将当前文档导出为 PDF 文件。
 *
 * @param options - 可覆盖所有扩展级选项
 * @example editor.commands.exportPdf({ onCompleteExport: (result) => {} })
 */
exportPdf: (options?: ExportPdfCommandOptions) => ReturnType
```

`ExportPdfCommandOptions` 接口继承了 `ExportPdfOptions`，因此每个配置选项都可以在命令调用时覆盖：

| Property           | Type                      | Description                                                                           |
| ------------------ | ------------------------- | ------------------------------------------------------------------------------------- |
| `endpoint`         | `string`                  | 针对该次导出覆盖 API 端点                                                             |
| `token`            | `string`                  | 针对该次导出覆盖 JWT 令牌                                                             |
| `appId`            | `string`                  | 针对该次导出覆盖 App ID                                                               |
| `styleOverrides`   | `Record<string, string>`  | 针对该次导出覆盖样式覆盖                                                              |
| `headers`          | `HeaderConfig`            | 针对该次导出覆盖页眉配置                                                              |
| `footers`          | `FooterConfig`            | 针对该次导出覆盖页脚配置                                                              |
| `pageSize`         | `PageSize`                | 针对该次导出覆盖页面大小                                                              |
| `pageMargins`      | `PageMargins`             | 针对该次导出覆盖页面边距                                                              |
| `customNodes`      | `CustomNodeDefinition[]`  | 针对该次导出覆盖自定义节点定义                                                        |
| `customFonts`      | `CustomFont[]`            | 针对该次导出覆盖自定义字体（仅本地部署）                                              |
| `onCompleteExport` | `(result: Blob) => void`  | 针对该次导出覆盖回调函数                                                              |

```js
import { ExportPdf } from '@tiptap-pro/extension-export-pdf'

const editor = new Editor({
  extensions: [
    // 其他扩展 ...
    ExportPdf.configure({
      token: 'YOUR_TOKEN',
      appId: 'YOUR_APP_ID',
      onCompleteExport(result) {
        // 下载 PDF 文件
        const url = URL.createObjectURL(result)
        const a = document.createElement('a')

        a.href = url
        a.download = 'document.pdf'
        a.click()

        URL.revokeObjectURL(url)
      },
    }),
    // 其他扩展 ...
  ],
})

// 使用扩展级回调
editor.chain().exportPdf().run()

// 或为特定导出覆盖选项
editor.chain().exportPdf({
  pageSize: { width: '8.5in', height: '11in' },
  onCompleteExport(result) {
    // 针对该导出自定义处理
    const url = URL.createObjectURL(result)
    window.open(url)
  },
}).run()
```

### 工作原理

调用 `exportPdf` 时，扩展将编辑器文档序列化为 JSON，并连同配置的选项（样式覆盖、页面布局、页眉、页脚、自定义字体）一起发送给 Tiptap 转换服务。服务将文档转换为 PDF，并以二进制 Blob 返回。`onCompleteExport` 回调直接接收该 Blob，您可以根据需要进行下载或处理。错误将作为异常抛出，而非通过回调传递。

## 自定义字体

<Callout title="仅限本地部署" variant="warning">
    自定义字体仅在本地部署中可用。如果您有兴趣在 Tiptap 云服务中使用自定义字体，请<a href="https://tiptap.dev/pricing">联系我们了解企业计划</a>。
</Callout>

PDF 导出支持本地部署的自定义字体。您可以通过提供字体 URL 数组来配置转换服务下载并在生成 PDF 时使用这些字体。

字体文件必须是 `.ttf` 或 `.woff2` 格式，且支持 HTTPS 访问。`fontFamily` 必须与文档文本样式中使用的字体名匹配。

```ts
ExportPdf.configure({
  token: 'YOUR_TOKEN',
  appId: 'YOUR_APP_ID',
  customFonts: [
    {
      url: 'https://your-cdn.com/fonts/CustomFont-Regular.ttf',
      fontFamily: 'Custom Font',
    },
    {
      url: 'https://your-cdn.com/fonts/CustomFont-Bold.ttf',
      fontFamily: 'Custom Font',
    },
  ],
  onCompleteExport(result) {
    // Handle the exported PDF
  },
})
```

您也可以在命令调用时覆盖自定义字体：

```ts
editor.chain().exportPdf({
  customFonts: [
    {
      url: 'https://your-cdn.com/fonts/SpecialFont.woff2',
      fontFamily: 'Special Font',
    },
  ],
}).run()
```
