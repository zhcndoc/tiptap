ODT 导出扩展支持自定义页面尺寸和页边距，允许您创建符合精确布局要求的文档。无论您需要 A5 页面、自定义纸张尺寸，还是特定的页边距配置，都可以直接在扩展中进行设置。

## 页面尺寸配置

使用 `pageSize` 选项定义导出 ODT 文件的自定义页面尺寸。页面尺寸配置接受带有多种计量单位的宽度和高度值。

### 支持的单位

所有页面尺寸和页边距单位支持以下单位：

| 单位     | 描述                |
|----------|---------------------|
| `cm`     | 厘米（默认）        |
| `in`     | 英寸                |
| `pt`     | 磅点                |
| `pc`     | 派卡                |
| `mm`     | 毫米                |
| `px`     | 像素                |

### 配置选项

| 属性      | 类型       | 描述                                                                                  | 默认值       |
|-----------|------------|---------------------------------------------------------------------------------------|--------------|
| `width`   | `string`   | 页面宽度。必须是正数，后接有效单位（cm、in、pt、pc、mm、px）。                          | `"21cm"`     |
| `height`  | `string`   | 页面高度。必须是正数，后接有效单位（cm、in、pt、pc、mm、px）。                          | `"29.7cm"`   |

```js
import { ExportOdt } from '@tiptap-pro/extension-export-odt'

ExportOdt.configure({
  token: 'YOUR_TOKEN',
  appId: 'YOUR_APP_ID',
  pageSize: {
    width: '14.8cm',    // A5 width
    height: '21cm',     // A5 height
  },
})
```

## 页面边距配置

`pageMargins` 选项允许您为文档页面的所有边设置自定义边距。与页面尺寸不同，上下边距可以接受负值。

### 配置选项

| 属性      | 类型       | 描述                                                                                      | 默认值    |
|-----------|------------|-------------------------------------------------------------------------------------------|----------|
| `top`     | `string`   | 页面顶部边距。可为负值。必须是数字，后接有效单位（cm、in、pt、pc、mm、px）。              | `"1cm"`  |
| `bottom`  | `string`   | 页面底部边距。可为负值。必须是数字，后接有效单位（cm、in、pt、pc、mm、px）。              | `"1cm"`  |
| `left`    | `string`   | 页面左侧边距。必须是正数，后接有效单位（cm、in、pt、pc、mm、px）。                       | `"1cm"`  |
| `right`   | `string`   | 页面右侧边距。必须是正数，后接有效单位（cm、in、pt、pc、mm、px）。                       | `"1cm"`  |

```js
import { ExportOdt } from '@tiptap-pro/extension-export-odt'

ExportOdt.configure({
  token: 'YOUR_TOKEN',
  appId: 'YOUR_APP_ID',
  pageMargins: {
    top: '2cm',
    bottom: '2cm',
    left: '1.5cm',
    right: '1.5cm',
  },
})
```

## 完整示例

以下是如何使用 ODT 导出扩展配置自定义页面布局的完整示例：

```js
import { ExportOdt } from '@tiptap-pro/extension-export-odt'

const editor = new Editor({
  extensions: [
    // Other extensions...
    ExportOdt.configure({
      token: 'YOUR_TOKEN',
      appId: 'YOUR_APP_ID',
      // A5 page size
      pageSize: {
        width: '14.8cm',
        height: '21cm',
      },
      // Custom margins
      pageMargins: {
        top: '2cm',
        bottom: '2cm',
        left: '2cm',
        right: '2cm',
      },
    }),
    // Other extensions...
  ],
})

// Export with custom layout
editor
  .chain()
  .exportOdt({
    onCompleteExport(result) {
      const url = URL.createObjectURL(result)
      const a = document.createElement('a')

      a.href = url
      a.download = 'custom-layout.odt'
      a.click()

      URL.revokeObjectURL(url)
    },
  })
  .run()
```

## 常见页面尺寸

以下是一些可供参考的常见页面尺寸：

| 格式       | 宽度      | 高度      |
|------------|-----------|-----------|
| **A4**     | `21cm`    | `29.7cm`  |
| **A5**     | `14.8cm`  | `21cm`    |
| **Letter** | `8.5in`   | `11in`    |
| **Legal**  | `8.5in`   | `14in`    |
| **Tabloid**| `11in`    | `17in`    |

## 不同的计量单位

您可以根据喜好混合使用不同单位：

```js
ExportOdt.configure({
  token: 'YOUR_TOKEN',
  appId: 'YOUR_APP_ID',
  // US Letter size (inches)
  pageSize: {
    width: '8.5in',
    height: '11in',
  },
  // Margins with different units
  pageMargins: {
    top: '0.75in',    // inches
    bottom: '72pt',   // points (1 inch = 72 points)
    left: '2cm',      // centimeters
    right: '20mm',    // millimeters
  },
})
```

<Callout title="单位一致性" variant="hint">
    虽然您可以混用不同单位，但通常建议在整个配置中保持单位一致，以便维护和理解。
</Callout>
