导出为 DOC 时，你可以定义自定义样式覆盖，这些样式会应用到导出的文档中。当你想对导出文件的外观和感觉进行超出默认样式的自定义时，这非常有用。

## 样式覆盖

`styleOverrides` 选项接受一个 `Record<string, string>` 对象，其中键是样式属性名，值是样式值。这些覆盖会发送到 Tiptap 转换服务，并在转换过程中应用。

| 参数              | 类型                     | 描述                                                  | 默认值       |
| ----------------- | ------------------------ | ----------------------------------------------------- | ------------ |
| `styleOverrides`  | `Record<string, string>` | 应用于导出 DOC 文档的样式覆盖                         | `undefined`  |

## 示例

```js
import { ExportDoc } from '@tiptap-pro/extension-export-doc'

const editor = new Editor({
  extensions: [
    // 其他扩展...
    ExportDoc.configure({
      token: 'YOUR_TOKEN',
      appId: 'YOUR_APP_ID',
      styleOverrides: {
        'font-family': 'Arial, sans-serif',
        'font-size': '12pt',
        'line-height': '1.5',
        'color': '#333333',
      },
    }),
    // 其他扩展...
  ],
})

// 使用自定义样式导出
editor
  .chain()
  .exportDoc({
    onCompleteExport(result) {
      const url = URL.createObjectURL(result)
      const a = document.createElement('a')

      a.href = url
      a.download = 'styled-document.doc'
      a.click()

      URL.revokeObjectURL(url)
    },
  })
  .run()
```
