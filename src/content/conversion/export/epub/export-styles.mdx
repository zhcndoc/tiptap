导出为 EPUB 时，您可以定义自定义样式覆盖，这些覆盖将应用于导出的文档。当您希望自定义导出的外观和样式，超出默认样式时，这非常有用。

## 样式覆盖

`styleOverrides` 选项接受一个 `Record<string, string>` 对象，其中键是样式属性名，值是样式值。这些覆盖将被发送到 Tiptap 转换服务，并在转换过程中应用。

| 参数             | 类型                     | 说明                                                     | 默认值       |
| ---------------- | ------------------------ | -------------------------------------------------------- | ------------ |
| `styleOverrides` | `Record<string, string>` | 应用于导出 EPUB 文档的样式覆盖                           | `undefined`  |

## 示例

```js
import { ExportEpub } from '@tiptap-pro/extension-export-epub'

const editor = new Editor({
  extensions: [
    // 其他扩展...
    ExportEpub.configure({
      token: 'YOUR_TOKEN',
      appId: 'YOUR_APP_ID',
      styleOverrides: {
        'font-family': 'Arial, sans-serif',
        'font-size': '12pt',
        'line-height': '1.5',
        'color': '#333333',
      },
    }),
    // 其他扩展...
  ],
})

// 使用自定义样式导出
editor
  .chain()
  .exportEpub({
    onCompleteExport(result) {
      const url = URL.createObjectURL(result)
      const a = document.createElement('a')

      a.href = url
      a.download = 'styled-document.epub'
      a.click()

      URL.revokeObjectURL(url)
    },
  })
  .run()
```
