[PageBreak 扩展](/pages/core-concepts/page-break-node)允许用户在编辑器中插入显式分页符。当与 EPUB 导出扩展结合使用时，这些分页符会包含在导出的文档中。

## 工作原理

EPUB 导出完全支持分页符。EPUB 导出在底层使用 DOCX 转换，其中 `pageBreak` 节点会被转换为 `<w:br w:type="page"/>`，从而在生成的 EPUB 文件中产生标准分页符。

## 配置 PageBreak 与 EPUB 导出

将 PageBreak 扩展与 EPUB 导出扩展一起注册。导出时，编辑器内容中的分页符将被自动转换。

```js
import { Editor } from '@tiptap/core'
import StarterKit from '@tiptap/starter-kit'
import { ExportEpub } from '@tiptap-pro/extension-export-epub'
import { PageBreak } from '@tiptap-pro/extension-pagebreak'

const editor = new Editor({
  extensions: [
    StarterKit,
    PageBreak,
    ExportEpub.configure({
      appId: 'your-app-id',
      token: 'your-jwt',
      onCompleteExport: (result) => {
        const blob = new Blob([result], { type: 'application/epub+zip' })
        const url = URL.createObjectURL(blob)
        const a = document.createElement('a')
        a.href = url
        a.download = 'export.epub'
        a.click()
        URL.revokeObjectURL(url)
      },
    }),
  ],
})
```
