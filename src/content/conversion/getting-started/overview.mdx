Tiptap 转换服务使在 Tiptap 与文档文件格式之间导入和导出内容变得轻而易举。它提供：

- **导入**：将文件（Word DOCX、OpenOffice/LibreOffice ODT 或 Markdown 文件）转换为 Tiptap 的 JSON 文档格式。
- **导出**：将 Tiptap JSON 转回 Word DOCX、OpenOffice/LibreOffice ODT 或 Markdown 文件。
- **编辑器扩展**：使用内置的 Tiptap Pro 扩展直接在 Tiptap 编辑器实例中进行导入/导出。
- **REST API**：通过 HTTP 端点在任何需要的地方集成我们的转换服务，无需使用 Tiptap 编辑器。
