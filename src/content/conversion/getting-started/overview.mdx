Tiptap 转换服务让 Tiptap 与文档文件格式之间的内容导入导出变得轻而易举。它提供：

- **导入**: 将文件转换为 Tiptap 的 JSON 文档格式：Word DOCX 或 Markdown。
- **导出**: 将 Tiptap JSON 转换为其他文档格式：Word DOCX、PDF、ODT (OpenDocument Text)、EPUB、DOC (旧版 Word) 或 Markdown。
- **编辑器扩展**: 使用内置的 Tiptap Pro 扩展，直接在 Tiptap 编辑器实例中导入/导出 DOCX 和 Markdown。
- **REST API**: 使用 HTTP 端点在任何需要的地方集成我们的转换服务，无需 Tiptap 编辑器。REST API 目前支持所有导入和导出格式。

<Callout title="Pro 套餐" variant="hint">
  转换功能是包含在所有 [Tiptap 订阅](https://tiptap.dev/pricing) 中的 Pro 套餐的一部分。在安装之前，您需要设置 [访问 Tiptap 私有 NPM 注册表](/guides/pro-extensions) 的权限。
</Callout>
