Tiptap 转换服务使在 Tiptap 和文档文件格式之间导入和导出内容变得容易。它提供：

<Callout title="集成或测试转换" variant="warning">
    要测试或集成此功能，您需要一个 [活动试用或 Tiptap 订阅](https://cloud.tiptap.dev/v2/billing)。
</Callout>

- **导入**：将文件（Word DOCX、OpenOffice/LibreOffice ODT 或 Markdown 文件）转换为 Tiptap 的 JSON 文档格式。
- **导出**：将 Tiptap JSON 转回 Word DOCX、OpenOffice/LibreOffice ODT 或 Markdown 文件。
- **编辑器扩展**：使用内置的 Tiptap Pro 扩展直接在 Tiptap 编辑器实例中进行导入/导出。
- **REST API**：使用 HTTP 端点在需要的任何地方集成我们的转换服务，而无需 Tiptap 编辑器。
