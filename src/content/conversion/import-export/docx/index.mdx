Integrating `DOCX` (Microsoft Word) conversion with Tiptap can be done in several ways, from adding in-editor “Import/Export DOCX” buttons to using the REST API for server-side workflows.

| 方法                      | 描述                                                                                                                                                                                     | 文档说明                                                                                                                                                 |
|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| **编辑器导入**             | 允许用户直接在编辑器中上传 `.docx` 文件（例如，“导入 DOCX”按钮），将其转换为 Tiptap JSON 进行编辑。此扩展会向我们的 REST API 发送请求进行转换。                                        | [DOCX 编辑器导入](/conversion/import-export/docx/editor-import)                                                                                      |
| **编辑器导出**             | 提供从 Tiptap 内容生成 `.docx` 的功能。此扩展可在客户端和服务器端使用，并支持 **自定义节点**。                                                                                       | [DOCX 编辑器导出](/conversion/import-export/docx/editor-export)                                                                                      |
| **REST API**               | 适用于服务器端或外部的 `.docx` 转换（批处理，无需在编辑器中交互）。发送 `.docx` 以获取 Tiptap JSON 或反之。 **无法处理自定义节点。**                                                | [DOCX 转换REST API](/conversion/import-export/docx/rest-api)                                                                                        |
| **自定义节点和样式**         | 如果你有自定义的 Tiptap 扩展或需要更深层次的样式控制，可以覆盖默认设置或为 `.docx` 输出提供自定义渲染器。                                                                                   | [导出 DOCX 样式](/conversion/import-export/docx/export-styles)，[自定义节点用于 DOCX](/conversion/import-export/docx/custom-node-conversion)             |

<Callout title="关于自定义节点的信息" variant="hint">
    REST API 无法处理自定义节点转换。为此，你可以在自己的服务器或客户端实现 Editor Export 扩展，以定义这些节点的转换方式。
</Callout>
