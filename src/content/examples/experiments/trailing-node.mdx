尾节点扩展会自动在文档末尾添加一个不可见的节点。这对于添加一致的页脚或签名等目的非常有用。

在此处显示的示例中，尾节点保持为空，以便您可以将光标放在代码块后面。

由于此扩展**尚未正式发布**，您可能需要复制源代码并创建自己的扩展。

### 配置行为
在此示例中，当在最后一个节点中输入文本时，系统不会自动创建新的尾节点。此行为是可配置的。相关配置可以在 addOptions() 方法中实现，如下所示：

```js
addOptions() {
    return {
        node: 'paragraph',
        notAfter: ['paragraph'],
    };
}
```
