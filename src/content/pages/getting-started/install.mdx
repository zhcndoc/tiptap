## 安装 Pages 扩展

```bash
npm install @tiptap-pro/extension-pages@1.0.0-alpha.16.2
```

## 集成 Pages 扩展

安装 `pages` 扩展后，通过将 `Pages` 扩展添加到编辑器的 `extensions` 数组中，在编辑器中注册该扩展。

```js
import { Pages } from '@tiptap-pro/extension-pages'

const editor = new Editor({
  ...
  extensions: [
    ...,
    Pages,
  ]
})
```

根据您的使用场景，您可能需要配置协作提供者或其他选项。请参考完整文档以获取高级配置说明。
