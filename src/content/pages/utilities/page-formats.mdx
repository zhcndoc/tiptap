`PAGE_FORMATS` 常量提供了一组预定义的页面格式（如 A4、A3、Letter 等），包含真实尺寸和推荐边距，且已转换为像素。这对于快速设置 Tiptap Pages 扩展中的 `pageFormat` 选项或自定义布局非常有用。

## 导入

```js
import { PAGE_FORMATS } from '@tiptap-pro/extension-pages'
```

## API

```ts
PAGE_FORMATS: {
  [format: string]: {
    id: string
    width: number // 单位：像素
    height: number // 单位：像素
    margins: {
      top: number
      right: number
      bottom: number
      left: number
    }
  }
}
```

## 可用格式

像素值使用标准 CSS 像素（1 英寸等于 96 像素）。这些常量在内部保留了小数像素精度；下方的整数为便于阅读而四舍五入后的结果。

| Format      | Width (px) | Height (px) | Margins (px: top / right / bottom / left) |
| ----------- | ---------- | ----------- | ----------------------------------------- |
| **A4**      | 794        | 1123        | 95 / 76 / 95 / 76                         |
| **A3**      | 1123       | 1587        | 95 / 76 / 95 / 76                         |
| **A5**      | 559        | 794         | 76 / 57 / 76 / 57                         |
| **Letter**  | 816        | 1056        | 96 / 96 / 96 / 96                         |
| **Legal**   | 816        | 1345        | 96 / 96 / 96 / 96                         |
| **Tabloid** | 1056       | 1633        | 96 / 96 / 96 / 96                         |

有关底层真实世界尺寸的厘米单位，请参阅 [Page format](/pages/core-concepts/page-format)。
