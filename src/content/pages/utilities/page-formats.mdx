`PAGE_FORMATS` 常量提供了一组预定义的页面格式（如 A4、A3、Letter 等），包含真实尺寸和推荐边距，且已转换为像素。这对于快速设置 Tiptap Pages 扩展中的 `pageFormat` 选项或自定义布局非常有用。

## 导入

```js
import { PAGE_FORMATS } from '@tiptap-pro/extension-pages'
```

## API

```ts
PAGE_FORMATS: {
  [format: string]: {
    id: string
    width: number // 单位：像素
    height: number // 单位：像素
    margins: {
      top: number
      right: number
      bottom: number
      left: number
    }
  }
}
```

## 可用格式

| 格式       | 宽度（像素） | 高度（像素） | 边距（像素：上 / 右 / 下 / 左）      |
|------------|--------------|--------------|--------------------------------------|
| **A4**      | 794          | 1123         | 95 / 76 / 95 / 76                    |
| **A3**      | 1123         | 1591         | 95 / 76 / 95 / 76                    |
| **A5**      | 419          | 794          | 76 / 57 / 76 / 57                    |
| **Letter**  | 818          | 1060         | 96 / 96 / 96 / 96                    |
| **Legal**   | 818          | 1404         | 96 / 96 / 96 / 96                    |
| **Tabloid** | 1060         | 1635         | 96 / 96 / 96 / 96                    |
