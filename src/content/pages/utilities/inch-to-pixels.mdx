`inchToPixels` 辅助函数让你可以轻松地将英寸转换为像素，这在你想用英寸定义页面或边距尺寸，并需要将它们转换为像素以用于 Tiptap Pages 扩展中的 `pageFormat` 选项或自定义布局时非常有用。

## 导入

```js
import { inchToPixels } from '@tiptap-pro/extension-pages'
```

## API

```ts
inchToPixels(inches: number, dpi?: number): number
```

- `inches`：要转换的英寸值。
- `dpi`（可选）：每英寸点数。如果未提供，该工具将自动检测当前环境的 DPI。
- **返回值：** 等同的像素值（四舍五入）。
