PageKit 是一个便捷的扩展，它将 Pages、Table 和 PageBreak 扩展打包在一起。它可以让你一次性配置这三个扩展，用于构建一个支持分页、具备表格功能且支持分页符的编辑器。

## 1. 安装 PageKit 扩展

```bash
npm install @tiptap-pro/extension-pages-pagekit
```

## 2. 导入并使用 PageKit

```js
import { Editor } from '@tiptap/core'
import StarterKit from '@tiptap/starter-kit'
import { PageKit } from '@tiptap-pro/extension-pages-pagekit'

const editor = new Editor({
  extensions: [
    StarterKit,
    PageKit.configure({
      pages: {
        pageFormat: 'A4',
        header: '我的项目',
        footer: '第 {page} 页，共 {total} 页',
      },
      table: {
        // 表格选项（可选）
        resizable: true,
      },
      pagebreak: {
        // 在此处设置 PageBreak 选项（可选）
        label: '分页符',
      },
    }),
  ],
})
```

<Callout title="工作原理" variant="info">
  PageKit 会为你配置 Pages、Table 和 PageBreak 扩展。在 <code>pages</code> 下传递 Pages 选项，
  在 <code>table</code> 下传递 Table 选项，并在 <code>pagebreak</code> 下传递 PageBreak 选项。
</Callout>

## 禁用 PageBreak

如果你不需要显式的分页符，可以禁用 PageBreak 扩展：

```js
PageKit.configure({
  pages: { pageFormat: 'A4' },
  pagebreak: false, // 禁用 PageBreak
})
```

## 3. 何时使用 PageKit？

- 当你同时需要分页和表格支持时，PageKit 是最快速的设置方式。
- 如果你需要对 Table 或 Pages 进行深入定制，建议分别导入并单独配置它们。
