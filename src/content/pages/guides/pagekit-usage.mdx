PageKit 是一个便捷的扩展包，集成了 Pages 和 Table 两个扩展。它让你能同时配置这两个扩展，打造支持分页和表格的编辑器。

## 1. 安装 Pages 扩展

```bash
npm install @tiptap-pro/extension-page-kit
```

## 2. 导入并使用 PageKit

```js
import { Editor } from '@tiptap/core'
import StarterKit from '@tiptap/starter-kit'
import { PageKit } from '@tiptap-pro/extension-page-kit'

const editor = new Editor({
  extensions: [
    StarterKit,
    PageKit.configure({
      pages: {
        pageFormat: 'A4',
        header: '我的项目',
        footer: '第 {page} 页，共 {total} 页',
      },
      table: {
        // 表格选项（可选）
        resizable: true,
      },
    }),
  ],
})
```

<Callout title="工作原理" variant="info">
  PageKit 会为你配置 Pages 和 Table 两个扩展。将 Pages 配置项放在{' '}
  <code>pages</code> 下，将 Table 配置项放在 <code>table</code> 下。
</Callout>

## 3. 何时使用 PageKit？

- 需要同时支持分页和表格时，使用 PageKit 可以最快速完成设置。
- 如果你需要对 Table 或 Pages 进行深入定制，建议分别导入并单独配置它们。
